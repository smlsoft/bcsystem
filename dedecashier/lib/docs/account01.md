## ภาษีมูลค่าเพิ่ม (Value Added Tax - VAT) ในประเทศไทย: ข้อมูลเชิงลึกและกฎหมายที่เกี่ยวข้อง

ข้อมูลต่อไปนี้คือรายละเอียดเชิงลึกเกี่ยวกับหลักเกณฑ์ของภาษีมูลค่าเพิ่ม (VAT) ในประเทศไทย ตามประมวลรัษฎากรและกฎหมายที่เกี่ยวข้อง

---

### 1. จุดความรับผิดในการเสียภาษี (Tax Point)

การกำหนดจุดความรับผิดในการเสีย VAT (Tax Point) เป็นสิ่งสำคัญที่บอกว่าผู้ประกอบการต้องออกใบกำกับภาษีเมื่อใด และนำภาษีขายมารวมคำนวณในเดือนภาษีใด

#### 1.1 การขายสินค้าทั่วไป (มาตรา 78(1))

ความรับผิดในการเสียภาษี (Tax Point) จะเกิดขึ้นเมื่อมีการ **ส่งมอบสินค้า** เป็นหลัก **ยกเว้น** กรณีที่เหตุการณ์ใดเหตุการณ์หนึ่งต่อไปนี้เกิดขึ้นก่อนการส่งมอบสินค้า ให้ถือว่าความรับผิดเกิดขึ้นทันทีที่เกิดเหตุการณ์นั้น ๆ:

* **โอนกรรมสิทธิ์สินค้า** (เช่น การจดทะเบียนโอนรถยนต์)
* **ได้รับชำระราคาสินค้า** (ทั้งหมดหรือบางส่วน)
* **ออกใบกำกับภาษี** (ทั้งหมดหรือบางส่วน)

#### 1.2 การให้บริการ (มาตรา 78/3)

ความรับผิดในการเสียภาษี (Tax Point) จะเกิดขึ้นเมื่อมีการ **ชำระราคาค่าบริการ** เป็นหลัก **ยกเว้น** กรณีที่เหตุการณ์ใดเหตุการณ์หนึ่งต่อไปนี้เกิดขึ้นก่อนการชำระราคา ให้ถือว่าความรับผิดเกิดขึ้นทันทีที่เกิดเหตุการณ์นั้น ๆ:

* **ได้ใช้บริการ** ตามส่วนของบริการที่ได้ใช้
* **ออกใบกำกับภาษี**

**ข้อสังเกต:** สำหรับกิจการค้าปลีก (ขายสินค้า/ให้บริการรายย่อย) ที่มีการขาย/ให้บริการที่มีมูลค่าครั้งหนึ่งไม่เกิน 1,000 บาท สามารถจัดทำใบกำกับภาษีอย่างย่อรวบรวมการขายในหนึ่งวันทำการได้ เว้นแต่ลูกค้าจะเรียกร้องใบกำกับภาษีเต็มรูป

---

### 2. อัตราภาษี 0% (Zero Rate VAT) (มาตรา 80/1)

กิจการที่เสีย VAT ในอัตราร้อยละ 0% หมายถึง ผู้ประกอบการสามารถเรียกเก็บ VAT 0% จากผู้ซื้อได้ แต่ยังคงมีสิทธินำ **ภาษีซื้อ** มาขอคืนได้ทั้งหมด ซึ่งแตกต่างจากการ **ยกเว้น VAT** ที่ไม่มีสิทธินำภาษีซื้อมาขอคืน

กิจการที่ใช้ VAT 0% ได้แก่:

1.  **การส่งออกสินค้า:** (ที่มิใช่การส่งออกที่ได้รับยกเว้น VAT)
2.  **การให้บริการที่กระทำในราชอาณาจักรและได้มีการใช้บริการนั้นในต่างประเทศ**
3.  **การให้บริการขนส่งระหว่างประเทศ** โดยอากาศยานหรือเรือเดินทะเลที่กระทำโดยนิติบุคคล
4.  **การขายสินค้าหรือให้บริการแก่หน่วยงานของรัฐ** ตามโครงการเงินกู้หรือเงินช่วยเหลือจากต่างประเทศ
5.  **การขายสินค้าหรือให้บริการแก่:** องค์การสหประชาชาติ, สถานเอกอัครราชทูต, สถานกงสุล หรือองค์การระหว่างประเทศ
6.  **การขายสินค้าระหว่างคลังสินค้าทัณฑ์บน** หรือ **เขตปลอดอากร**

---

### 3. สินค้าและบริการที่ได้รับยกเว้นภาษีมูลค่าเพิ่ม (VAT Exemption) (มาตรา 81)

กิจการที่ได้รับยกเว้น VAT จะ **ไม่มีสิทธินำภาษีซื้อ** มาหักออกจากภาษีขาย หรือขอคืนภาษีซื้อได้

| หมวดหมู่ | รายละเอียด |
| :--- | :--- |
| **รายรับขั้นต่ำ** | การขายสินค้าหรือให้บริการของผู้ประกอบการที่มีรายรับไม่เกิน **1.8 ล้านบาทต่อปี** (มีสิทธิเลือกจด VAT ได้) |
| **กลุ่มสินค้าเกษตร** | การขายพืชผลทางการเกษตร สัตว์ (มีชีวิต/ไม่มีชีวิต) ภายในประเทศ, ปุ๋ย, ปลาป่น, อาหารสัตว์, ยา/เคมีภัณฑ์สำหรับพืชหรือสัตว์ (เพื่อบำรุง, ป้องกัน, กำจัดศัตรู/โรค) |
| **กลุ่มสิ่งพิมพ์** | การขายหนังสือพิมพ์ นิตยสาร หรือตำราเรียน |
| **กลุ่มบริการพื้นฐาน** | บริการขนส่งในราชอาณาจักร (ยกเว้นทางอากาศยานและทางท่อ), บริการเช่าอสังหาริมทรัพย์ |
| **กลุ่มบริการเพื่อสังคม** | บริการรักษาพยาบาลของสถานพยาบาล, บริการด้านการศึกษา, บริการประกอบโรคศิลปะ การสอบบัญชี การว่าความ, บริการห้องสมุด พิพิธภัณฑ์ สวนสัตว์ |
| **กลุ่มการเงิน** | การประกันชีวิต, การให้บริการของสถาบันการเงิน |
| **กลุ่มรัฐบาล** | การขายสลากกินแบ่งของรัฐบาล, การขายแสตมป์ไปรษณีย์ในราคาที่ไม่เกินมูลค่าที่ตราไว้ |

### 4. ภาษีซื้อต้องห้าม (Non-creditable Input Tax) (มาตรา 82/5 และ พ.ร.ฎ. ฉบับที่ 243)

ภาษีซื้อบางประเภทไม่สามารถนำมาใช้เป็นเครดิตภาษีเพื่อหักออกจากภาษีขาย หรือขอคืนได้ ถือเป็น "ภาษีซื้อต้องห้าม" ซึ่งผู้ประกอบการต้องนำไปรวมเป็นต้นทุนสินค้า/ค่าใช้จ่าย

รายการภาษีซื้อต้องห้ามที่สำคัญ ได้แก่:

* **ภาษีซื้อที่ไม่มีใบกำกับภาษี:** หรือไม่สามารถแสดงใบกำกับภาษีได้ (เช่น ใบกำกับภาษีสูญหาย)
* **ใบกำกับภาษีที่ไม่สมบูรณ์:** ไม่ระบุชื่อที่อยู่ผู้ซื้อ/ผู้ขาย/เลขประจำตัวผู้เสียภาษี/วันที่ออก/มูลค่าสินค้าและ VAT ให้ชัดเจน
* **ใบกำกับภาษีที่ออกโดยผู้ไม่มีสิทธิออก:** เช่น ผู้ขายไม่ได้จดทะเบียน VAT แต่มีการออกใบกำกับภาษี
* **ภาษีซื้อที่เกิดจากการซื้อ/เช่าซื้อ/เช่ารถยนต์นั่ง:** และรถยนต์โดยสารที่มีที่นั่งไม่เกิน 10 คน (เช่น รถเก๋ง, รถตู้ VIP)
* **ภาษีซื้อที่ไม่เกี่ยวข้องโดยตรงกับกิจการ:** เช่น การซื้อสินค้าเพื่อการส่วนตัวของผู้ประกอบการ
* **ภาษีซื้อที่เกิดจากค่ารับรอง:** หรือค่าบริการที่มีลักษณะทำนองเดียวกัน (ตามเงื่อนไขของภาษีเงินได้นิติบุคคล)

---

### 5. หน้าที่ของผู้ประกอบการจดทะเบียน VAT

ผู้ประกอบการที่จดทะเบียน VAT มีหน้าที่ตามกฎหมายที่ต้องปฏิบัติอย่างเคร่งครัด:

1.  **จัดทำใบกำกับภาษี:** ออกใบกำกับภาษีให้ผู้ซื้อ/ผู้รับบริการทุกครั้งที่เกิด Tax Point และเก็บสำเนาไว้
2.  **จัดทำรายงาน:** จัดทำ **รายงานภาษีขาย** และ **รายงานภาษีซื้อ** รวมถึง **รายงานสินค้าและวัตถุดิบ** (ถ้ามี)
3.  **ยื่นแบบ ภ.พ.30:** ยื่นแบบแสดงรายการภาษีมูลค่าเพิ่ม (ภ.พ.30) และชำระภาษี (ถ้ามี) เป็นรายเดือนภาษี **ภายในวันที่ 15 ของเดือนถัดไป** (ต้องยื่นแม้จะไม่มีรายการซื้อขายในเดือนนั้นก็ตาม)
4.  **เก็บรักษาเอกสาร:** เก็บรักษาบัญชีและเอกสารประกอบการลงบัญชี รวมถึงใบกำกับภาษี (ซื้อและขาย) ไว้เป็นเวลาไม่น้อยกว่า 5 ปี

**บทลงโทษที่สำคัญ (กรณีไม่ปฏิบัติตาม):**

* **ไม่ชำระภาษี/ชำระเกินกำหนด:** ต้องเสีย **เงินเพิ่ม** อีกร้อยละ **1.5 ต่อเดือน** ของเงินภาษีที่ต้องชำระ
* **ไม่จัดทำใบกำกับภาษี/ไม่ส่งมอบใบกำกับภาษีทันที:** ต้องรับผิดชอบ **เบี้ยปรับ** และอาจมีโทษทางอาญา (จำคุก/ปรับ)
