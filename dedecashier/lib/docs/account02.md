## การเฉลี่ยภาษีขาย กรณีสินค้าเกษตรขายร่วมกับสินค้าทั่วไป และมีส่วนลดท้ายบิล

เมื่อผู้ประกอบการมีการขายสินค้าหลายประเภทในใบเสร็จเดียวกัน โดยมีทั้ง **สินค้าที่ต้องเสีย VAT 7% (สินค้าทั่วไป)** และ **สินค้าที่ได้รับยกเว้น VAT (สินค้าเกษตรที่ไม่แปรรูป)** และมีการให้ **ส่วนลดรวม (ส่วนลดท้ายบิล)** ผู้ประกอบการต้องทำการปันส่วน (เฉลี่ย) ส่วนลดดังกล่าวเพื่อนำมาคำนวณฐานภาษีมูลค่าเพิ่มให้ถูกต้อง

หลักการคำนวณที่ถูกต้องจะขึ้นอยู่กับหลักเกณฑ์ของ **ฐานภาษีมูลค่าเพิ่ม** และ **การนำส่วนลดมาหัก** ตามมาตรา 79 แห่งประมวลรัษฎากร

### 1. หลักการสำคัญของฐานภาษีและส่วนลด

| หลักการ | รายละเอียด |
| :--- | :--- |
| **ฐานภาษี VAT** | มูลค่าของฐานภาษีที่ใช้คำนวณ VAT 7% คือ **มูลค่าของสินค้าหรือบริการที่ต้องเสียภาษี (ก่อนรวม VAT) หักด้วยส่วนลดหรือค่าลดหย่อนที่ผู้ประกอบการจดทะเบียนให้แก่ผู้ซื้อ** (มาตรา 79) |
| **สินค้ายกเว้น VAT** | **สินค้าเกษตรที่ไม่แปรรูป** (เช่น ข้าว, ผักสด, ผลไม้สด) ได้รับการยกเว้น VAT ตามมาตรา 81 (ผู้ประกอบการไม่ต้องเรียกเก็บ VAT) |
| **ส่วนลดท้ายบิล** | ส่วนลดรวมที่ให้ท้ายบิล ต้องนำมา **ปันส่วน (เฉลี่ย)** ให้กับสินค้าแต่ละรายการ (ทั้งรายการที่เสีย VAT และยกเว้น VAT) ก่อนคำนวณภาษีขาย |

### 2. ขั้นตอนการคำนวณภาษีขาย (การปันส่วนส่วนลด)

สมมติให้มีการขายสินค้า 2 ประเภท และให้ส่วนลดท้ายบิลรวม **50 บาท**

| รายการสินค้า | ราคาขายปกติ (ไม่รวม VAT) | สถานะ VAT |
| :--- | :--- | :--- |
| **(ก) สินค้าทั่วไป (เสื้อผ้า)** | 500 บาท | เสีย VAT 7% |
| **(ข) สินค้าเกษตร (ผักสด)** | 300 บาท | ยกเว้น VAT (Non-VAT) |
| **รวมราคาขายปกติ** | **800 บาท** | |

#### ขั้นตอนที่ 1: คำนวณอัตราส่วนของสินค้าแต่ละประเภท

นำมูลค่าสินค้าแต่ละรายการมาหาอัตราส่วนต่อยอดรวม:

1.  **สัดส่วนสินค้าทั่วไป (VAT 7%):** $500 \text{ บาท} / 800 \text{ บาท} = 0.625$ (หรือ 62.5%)
2.  **สัดส่วนสินค้าเกษตร (ยกเว้น VAT):** $300 \text{ บาท} / 800 \text{ บาท} = 0.375$ (หรือ 37.5%)

#### ขั้นตอนที่ 2: ปันส่วนส่วนลดท้ายบิล

นำส่วนลดท้ายบิลรวม (**50 บาท**) มาปันส่วนตามสัดส่วนที่คำนวณได้:

1.  **ส่วนลดที่ปันส่วนให้สินค้าทั่วไป:** $50 \text{ บาท} \times 0.625 = 31.25$ บาท
2.  **ส่วนลดที่ปันส่วนให้สินค้าเกษตร:** $50 \text{ บาท} \times 0.375 = 18.75$ บาท
    * *(รวมส่วนลดปันส่วน: $31.25 + 18.75 = 50$ บาท)*

#### ขั้นตอนที่ 3: คำนวณฐานภาษีและภาษีขาย

นำราคาสินค้าแต่ละรายการ หักด้วยส่วนลดที่ปันส่วนได้ เพื่อหา **ฐานภาษี** (สำหรับรายการที่ต้องเสีย VAT)

1.  **รายการสินค้าเกษตร (ยกเว้น VAT):**
    * ราคาหลังหักส่วนลด: $300 \text{ บาท} - 18.75 \text{ บาท} = 281.25$ บาท
    * **ภาษีขาย:** 0 บาท (ไม่ต้องเสีย VAT)
2.  **รายการสินค้าทั่วไป (ต้องเสีย VAT 7%):**
    * **ฐานภาษี (ราคาหลังหักส่วนลด):** $500 \text{ บาท} - 31.25 \text{ บาท} = 468.75$ บาท
    * **ภาษีขาย (VAT 7%):** $468.75 \text{ บาท} \times 7\% = 32.81$ บาท

#### สรุปยอดรวมสำหรับใบเสร็จ

| รายการ | มูลค่าก่อน VAT/ยกเว้น | ภาษีขาย |
| :--- | :--- | :--- |
| สินค้าทั่วไป (ฐานภาษี) | $468.75$ บาท | $32.81$ บาท |
| สินค้าเกษตร (ยกเว้น VAT) | $281.25$ บาท | $0.00$ บาท |
| **ยอดรวมที่ต้องชำระ (ราคารวม VAT)** | **$750.00$ บาท** | **$32.81$ บาท** |
| **ราคาสุทธิที่ลูกค้าต้องจ่าย** | $750.00 \text{ บาท} + 32.81 \text{ บาท} = 782.81$ บาท | |

### 3. ข้อควรระวังและการจัดการทางบัญชี

1.  **ใบกำกับภาษี:** ผู้ประกอบการต้องระบุรายการและมูลค่าที่ถูกต้อง โดยเฉพาะการแยกแสดงมูลค่าที่ต้องเสีย VAT และมูลค่าที่ได้รับยกเว้น VAT อย่างชัดเจน
2.  **ภาษีซื้อเฉลี่ย (สำคัญ):** หากผู้ประกอบการมีทั้งรายได้ที่ต้องเสีย VAT และรายได้ที่ได้รับยกเว้น VAT ผู้ประกอบการต้องทำการ **เฉลี่ยภาษีซื้อ** (Apportionment of Input Tax) สำหรับค่าใช้จ่ายส่วนกลางที่ไม่สามารถระบุได้ว่าใช้สำหรับกิจการใดโดยเฉพาะ (เช่น ค่าเช่าร้าน, ค่าน้ำ/ค่าไฟ) โดยนำภาษีซื้อไปหักจากภาษีขายตามสัดส่วนของรายได้ที่ต้องเสีย VAT
3.  **การปัดเศษ:** การคำนวณและปัดเศษทศนิยมของ VAT ที่ต้องแสดงในใบกำกับภาษีจะต้องเป็นไปตามหลักการที่กรมสรรพากรกำหนด เพื่อให้สอดคล้องกับการลงบัญชี
