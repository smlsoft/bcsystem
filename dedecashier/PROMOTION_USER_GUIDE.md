# 📚 คู่มือการใช้งานระบบโปรโมชั่น (Promotion System User Guide)

## 🎯 ภาพรวมระบบโปรโมชั่น

ระบบโปรโมชั่นของ DedeCashier รองรับโปรโมชั่นหลากหลายรูปแบบ โดย**ระบบจะตรวจสอบทั้งรหัสสินค้า (Item Code) และอัตราส่วนหน่วยนับ (Unit Ratio)** เพื่อให้แน่ใจว่าโปรโมชั่นถูกนำไปใช้กับสินค้าที่ถูกต้อง

---

## ⚠️ **สำคัญมาก: เรื่องอัตราส่วนหน่วยนับ**

### 🔍 ระบบจะเช็คอัตราส่วนด้วย!

เมื่อตั้งโปรโมชั่น ระบบจะตรวจสอบ 2 สิ่ง:
1. **รหัสสินค้า (Item Code)** - ต้องตรงกัน
2. **อัตราส่วนหน่วยนับ (Unit Ratio)** - ต้องตรงกันด้วย!

#### ตัวอย่าง:
```
สินค้า: น้ำดื่ม COLA
- Barcode A: ขวดเดี่ยว (อัตราส่วน 1:1)
- Barcode B: กล่อง 12 ขวด (อัตราส่วน 12:1)
- Barcode C: ลัง 24 ขวด (อัตราส่วน 24:1)

⭐ แม้จะเป็นสินค้าเดียวกัน แต่ถ้าอัตราส่วนต่างกัน = โปรโมชั่นจะไม่ใช้ได้!
```

### ✅ กรณีตัวอย่าง:

**โปรโมชั่น**: ซื้อน้ำดื่ม **กล่อง (12:1)** ครบ 10 กล่อง ราคา 100 บาท/กล่อง

| สินค้าในตะกร้า | อัตราส่วน | ได้โปรโมชั่นหรือไม่ | เหตุผล |
|---------------|----------|-----------------|--------|
| ขวดเดี่ยว | 1:1 | ❌ **ไม่ได้** | อัตราส่วนไม่ตรงกับโปร |
| กล่อง 12 ขวด | 12:1 | ✅ **ได้** | อัตราส่วนตรงกับโปร |
| ลัง 24 ขวด | 24:1 | ❌ **ไม่ได้** | อัตราส่วนไม่ตรงกับโปร |

## 📋 ประเภทโปรโมชั่นทั้งหมด

### **Type 1: แถมสินค้าเมื่อซื้อครบตามจำนวน** 🎁

**คำอธิบาย**: ซื้อสินค้าครบตามจำนวนที่กำหนด แถมสินค้าฟรี

**เงื่อนไข**:
- เช็ครหัสสินค้า (Item Code) ✅
- เช็คอัตราส่วน (Unit Ratio) ✅
- เช็คจำนวนสินค้าที่ซื้อ ✅

**ตัวอย่าง**:
```
โปรโมชั่น: ซื้อ น้ำดื่ม (ลัง 24 ขวด) 5 ลัง แถม 1 ลัง

✅ ซื้อลัง (24:1) 5 ลัง → ได้แถม 1 ลัง
❌ ซื้อกล่อง (12:1) 10 กล่อง → ไม่ได้แถม (อัตราส่วนไม่ตรง)
❌ ซื้อขวดเดี่ยว (1:1) 120 ขวด → ไม่ได้แถม (อัตราส่วนไม่ตรง)
```

**หมายเหตุ**: 
- สินค้าที่แถมต้องมีในตะกร้าด้วย
- สินค้าที่แถมต้องมีอัตราส่วนตรงกับที่กำหนดไว้ในโปรโมชั่น

---

### **Type 2: ส่วนลดเงินสด/เปอร์เซ็นต์เมื่อซื้อครบตามจำนวน** 💰

**คำอธิบาย**: ซื้อสินค้าครบตามจำนวนที่กำหนด ได้ส่วนลด

**เงื่อนไข**:
- เช็ครหัสสินค้า (Item Code) ✅
- เช็คอัตราส่วน (Unit Ratio) ✅
- เช็คจำนวนสินค้าที่ซื้อ ✅

**ตัวอย่าง**:
```
โปรโมชั่น: ซื้อ ปูนซีเมนต์ (ถุง 50 กก.) 100 ถุง ลด 10%

✅ ซื้อถุง 50 กก. (50:1) 100 ถุง → ลด 10%
❌ ซื้อถุง 25 กก. (25:1) 200 ถุง → ไม่ลด (อัตราส่วนไม่ตรง)
```

**รูปแบบส่วนลด**:
- `%XX` = ลด XX เปอร์เซ็นต์ (เช่น %10 = ลด 10%)
- `XX` = ลด XX บาท (เช่น 50 = ลด 50 บาท)

---

### **Type 3: ซื้อสินค้าตาม List แถมสินค้าตาม List** 🛍️

**คำอธิบาย**: ซื้อสินค้าในกลุ่มที่กำหนด แถมสินค้าในกลุ่มที่กำหนด (เช่น ซื้อ 2 แถม 1)

**เงื่อนไข**:
- เช็ครหัสสินค้า (Item Code) ✅
- เช็คอัตราส่วน (Unit Ratio) ✅
- สินค้าต้องอยู่ใน List ที่กำหนด ✅
- ราคาต้องเท่ากัน ✅

**ตัวอย่าง**:
```
โปรโมชั่น: ซื้อกระเบื้อง (กล่อง 10 ผืน) ในกลุ่ม A 
           จำนวน 2 กล่อง แถม 1 กล่อง

✅ ซื้อกระเบื้องลาย 1 + ลาย 2 (กล่อง 10 ผืน) 2 กล่อง 
   → แถม 1 กล่อง
❌ ซื้อกระเบื้องผืนเดี่ยว (1:1) 20 ผืน 
   → ไม่แถม (อัตราส่วนไม่ตรง)
```

---

### **Type 4: ราคาพิเศษเมื่อซื้อครบตามจำนวน** 🏷️

**คำอธิบาย**: ซื้อสินค้าครบตามจำนวนที่กำหนด ได้ราคาพิเศษ

**เงื่อนไข**:
- เช็ครหัสสินค้า (Item Code) ✅
- **เช็คอัตราส่วน (Unit Ratio) ✅✅✅ สำคัญมาก!**
- เช็คจำนวนสินค้าที่ซื้อ ✅

**ตัวอย่าง**:
```
โปรโมชั่น: ซื้อ เหล็กเส้น (มัด 10 เส้น) ครบ 100 มัด 
           ราคา 180 บาท/มัด (ปกติ 220 บาท)

✅ ซื้อมัด (10:1) 100 มัด 
   → ราคา 180 บาท/มัด ✅
   
❌ ซื้อเส้นเดี่ยว (1:1) 1,000 เส้น 
   → ราคา 220 บาท/เส้น ❌ (อัตราส่วนไม่ตรง)
   
❌ ซื้อมัด (5:1) 200 มัด 
   → ราคาปกติ ❌ (อัตราส่วนไม่ตรง)
```

**หมายเหตุสำคัญ**:
- ราคาพิเศษจะใช้ได้เฉพาะหน่วยนับที่อัตราส่วนตรงกับโปรโมชั่นเท่านั้น!
- ถ้าซื้อหลายหน่วยนับ แต่ละหน่วยนับจะได้ราคาตามเงื่อนไขของตัวเอง

### **Type 5: ซื้อครบจำนวน ลดเปอร์เซ็นต์หรือบาท** 💸

**คำอธิบาย**: ซื้อสินค้าครบตามจำนวนที่กำหนด ลดราคา

**เงื่อนไข**:
- เช็ครหัสสินค้า (Item Code) ✅
- **เช็คอัตราส่วน (Unit Ratio) ✅✅✅**
- เช็คจำนวนสินค้าที่ซื้อ ✅

**ตัวอย่าง**:
```
โปรโมชั่น: ซื้อ กระเบื้องหลังคา (แพ็ค 100 แผ่น) 
           ครบ 10 แพ็ค ลด 15%

✅ ซื้อแพ็ค (100:1) 10 แพ็ค → ลด 15% ✅
❌ ซื้อแผ่นเดี่ยว (1:1) 1,000 แผ่น → ไม่ลด ❌
```

### **Type 6: ซื้อครบ XXX บาท แถมสินค้า** 💵

**คำอธิบาย**: ซื้อสินค้าในกลุ่มที่กำหนดครบตามยอดเงิน แถมสินค้า

**เงื่อนไข**:
- เช็ครหัสสินค้า (Item Code) ของสินค้าที่ต้องซื้อ ✅
- เช็คอัตราส่วน (Unit Ratio) ของสินค้าที่ต้องซื้อ ✅
- **เช็คอัตราส่วน (Unit Ratio) ของสินค้าที่จะแถม ✅✅✅**
- รวมยอดเงินจากสินค้าที่ตรงเงื่อนไข ✅

**ตัวอย่าง**:
```
โปรโมชั่น: ซื้อสินค้าในกลุ่ม A ครบ 1,000 บาท 
           แถม น้ำยาล้างจาน (ขวด 500 มล.)

เงื่อนไข:
- สินค้าที่นับยอด: ต้องเป็นกลุ่ม A และอัตราส่วนตรง
- สินค้าที่แถม: ต้องเป็นขวด 500 มล. (อัตราส่วนตรง)

✅ ซื้อสินค้า A (ถูกอัตราส่วน) 1,000 บาท 
   + มีน้ำยาขวด 500 มล. ในตะกร้า 
   → ได้แถม ✅
   
❌ ซื้อสินค้า A 1,000 บาท 
   + มีแต่น้ำยาขวด 1 ลิตร ในตะกร้า 
   → ไม่แถม ❌ (อัตราส่วนไม่ตรง)
```

### **Type 101: ซื้อครบ XXX บาท แลกคูปองส่วนลด (House Brand)** 🎟️

**คำอธิบาย**: ซื้อสินค้า House Brand ครบตามยอดเงิน แลกคูปองส่วนลด

**เงื่อนไข**:
- เช็ครหัสสินค้าประเภท House Brand (Pattern Code = "HB") ✅
- ไม่เช็คอัตราส่วน (รวมทุกหน่วยนับ)
- รวมยอดเงินจากสินค้า HB ทั้งหมด ✅

**ตัวอย่าง**:
```
โปรโมชั่น: ซื้อสินค้า House Brand ครบ 2,000 บาท 
           แลกคูปองลด 200 บาท

✅ ซื้อสินค้า HB (ทุกหน่วยนับ) รวม 2,000 บาท 
   → แลกคูปอง 1 ใบ (ลด 200 บาท) ✅
```

**หมายเหตุ**: Type 101 เป็น Type พิเศษที่ไม่เช็คอัตราส่วน เพราะเป็นการรวมยอดซื้อทั้งหมดของสินค้า HB

### **Type 102: Tier-based Redemption (ซื้อครบยอดแลกของรางวัล)** 🏆

**คำอธิบาย**: ซื้อสินค้าในกลุ่มที่กำหนดครบตามยอดเงิน แลกของรางวัลตามชั้น (Tier)

**เงื่อนไข**:
- แต่ละ Tier มี CSV ของสินค้าที่นับยอด ✅
- เช็ครหัสสินค้า (Item Code) ✅
- เช็คอัตราส่วน (Unit Ratio) ✅
- แต่ละบิลแลกได้แค่ 1 Tier เท่านั้น ✅

**ตัวอย่าง**:
```
โปรโมชั่น: ซื้อกระเบื้อง
- Tier 1: ซื้อ 5,000 บาท → แลกแก้วน้ำ
- Tier 2: ซื้อ 10,000 บาท → แลกกระติกน้ำ
- Tier 3: ซื้อ 20,000 บาท → แลกเครื่องครัว

✅ ซื้อกระเบื้อง (ตรงเงื่อนไข) 15,000 บาท 
   → แลก Tier 2 ได้ (กระติกน้ำ) ✅
   
⚠️ แลกได้แค่ 1 Tier ต่อ 1 บิล!
```

## 🎓 เทคนิคการใช้งาน

### ✅ เคล็ดลับ 1: ตรวจสอบอัตราส่วนก่อนตั้งโปร

```
ขั้นตอน:
1. ดูรายการ Barcode ของสินค้า
2. ดูอัตราส่วนแต่ละ Barcode (เช่น 1:1, 12:1, 24:1)
3. ตั้งโปรโมชั่นให้ตรงกับอัตราส่วนที่ต้องการ
```

### ✅ เคล็ดลับ 2: การตั้งโปรหลายหน่วยนับ

```
ถ้าต้องการให้ทุกหน่วยนับได้โปร:
- ต้องตั้งโปรแยกสำหรับแต่ละอัตราส่วน

ตัวอย่าง:
- โปร A: สำหรับขวดเดี่ยว (1:1)
- โปร B: สำหรับกล่อง (12:1)
- โปร C: สำหรับลัง (24:1)
```

### ✅ เคล็ดลับ 3: ตรวจสอบสินค้าในตะกร้า

```
ก่อนคาดหวังว่าจะได้โปร:
1. ตรวจสอบรหัสสินค้าถูกต้อง ✅
2. ตรวจสอบอัตราส่วนตรงกัน ✅
3. ตรวจสอบจำนวนครบ ✅
```

## 🚨 ข้อควรระวัง

### ⚠️ 1. Barcode ซ้ำในอัตราส่วนเดียวกัน

**ปัญหา**: บางร้านมี Barcode หลายอัน แต่เป็น Item Code เดียวกันและอัตราส่วนเดียวกัน

**วิธีแก้**: ระบบจะรวมจำนวนจาก Barcode ทั้งหมดที่มี Item Code และอัตราส่วนเดียวกัน

**ตัวอย่าง**:
```
Item: น้ำดื่ม COLA
- Barcode 1234: กล่อง 12 ขวด (12:1)
- Barcode 5678: กล่อง 12 ขวด (12:1)  ← ซ้ำกัน!

โปร: ซื้อกล่อง (12:1) 10 กล่อง ราคา 100 บาท

ถ้าซื้อ:
- Barcode 1234 จำนวน 5 กล่อง
- Barcode 5678 จำนวน 5 กล่อง

✅ รวมกัน = 10 กล่อง → ได้โปร!
```

### ⚠️ 2. สินค้าแถมต้องมีในตะกร้า

สำหรับโปร Type 1 (แถมสินค้า):
- สินค้าที่จะแถมต้องมีในตะกร้าก่อน
- อัตราส่วนของสินค้าที่แถมต้องตรงกับที่กำหนดในโปร

### ⚠️ 3. โปรโมชั่นหมดอายุ

- ระบบจะแสดงเตือนถ้าโปรหมดอายุหรือยังไม่เริ่ม
- สีแดง = หมดอายุแล้ว
- สีน้ำเงิน = ยังไม่ถึงวันที่เริ่ม

### ⚠️ 4. ยอดไม่ครบ

- ระบบจะแสดงเตือนว่าต้องซื้อเพิ่มอีกเท่าไร
- สีน้ำเงิน = กำลังจะได้โปร (ซื้อเพิ่มอีกนิด)
