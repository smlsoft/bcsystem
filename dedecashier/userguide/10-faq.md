## 🔐 การเข้าสู่ระบบ

### Q: ลืมรหัสผ่านพนักงานทำอย่างไร?
**A:** 
1. ติดต่อผู้จัดการหรือ Admin
2. ใช้รหัส Master Password รีเซ็ต
3. หรือเข้าระบบด้วยบัญชี Admin แล้วรีเซ็ตให้

### Q: หน้าจอ Login ไม่แสดง?
**A:** 
- กด **F5** รีเฟรชโปรแกรม
- ปิดโปรแกรมแล้วเปิดใหม่
- ตรวจสอบการเชื่อมต่ออินเทอร์เน็ต

### Q: Login ได้แต่ไม่เจอข้อมูลสินค้า?
**A:** 
- เช็คว่าเลือก**สาขาถูกต้อง**หรือไม่
- กดปุ่ม **"ซิงค์ข้อมูล"** (F11)
- รอให้ระบบดาวน์โหลดข้อมูลเสร็จ

## 🛒 การขายสินค้า

### Q: สินค้าที่สแกนไม่ออกรายการ?
**A:** ตรวจสอบตามลำดับ:
1. **บาร์โค้ดถูกต้องไหม?** (ลองพิมพ์ด้วยมือ)
2. **สินค้ามีในระบบไหม?** (ค้นหาด้วยชื่อ)
3. **สต็อกเหลือไหม?** (ดูในรายการสินค้า)
4. **สาขาถูกไหม?** (บางสินค้าขายเฉพาะสาขา)

### Q: สแกนสินค้าซ้ำทำอย่างไร?
**A:**
- **ลบรายการ:** กดที่รายการ → กด **Del** หรือ **ถังขยะ**
- **แก้จำนวน:** กดที่รายการ → แก้ไขจำนวน

### Q: เปลี่ยนราคาขายได้ไหม?
**A:**
- **ต้องมีสิทธิ์** แก้ไขราคา (Admin/Manager)
- กดที่รายการ → **แก้ไขราคา** → ยืนยันด้วยรหัสผ่าน

### Q: ขายสินค้าผ่านเครื่องชั่งได้ไหม?
**A:**
- ได้! สำหรับสินค้าที่ขายเป็นกิโลกรัม/กรัม
- ชั่งสินค้า → พิมพ์สติกเกอร์ → สแกนสติกเกอร์

## 📦 บาร์โค้ดและสแกนเนอร์

### Q: เครื่องสแกนบาร์โค้ดไม่ทำงาน?
**A:** ตรวจสอบ:
1. ✅ **สายเสียบสนิทไหม?** (USB/Bluetooth)
2. ✅ **มีไฟแสดงไหม?** (ตรวจสอบไฟ LED)
3. ✅ **สแกนบาร์โค้ดทดสอบได้ไหม?**
4. ✅ **โปรแกรมรับสัญญาณไหม?** (ทดสอบพิมพ์ใน Notepad)

**วิธีแก้:**
- ลองเสียบ **USB Port อื่น**
- รีสตาร์ทเครื่องสแกน (ถอดสายแล้วเสียบใหม่)
- สแกนบาร์โค้ด **Reset** ในคู่มือเครื่อง

### Q: สแกนแล้วขึ้นตัวอักษรแปลกๆ?
**A:**
- เครื่องสแกนอาจตั้งค่าเป็น **Keyboard ภาษาอื่น**
- สแกนบาร์โค้ด **"US Keyboard Mode"** ในคู่มือ
- หรือเปลี่ยนภาษาคีย์บอร์ด Windows เป็น **English (US)**

### Q: สแกนบาร์โค้ดไทยไม่ได้?
**A:**
- ระบบรองรับบาร์โค้ด**ตัวเลขเท่านั้น**
- ถ้าต้องการใช้อักษรไทย → **พิมพ์ชื่อสินค้าแทน**

### Q: ใช้มือถือสแกนบาร์โค้ดได้ไหม?
**A:**
- **ได้!** ใช้ **QR Scanner** บนมือถือ
- เชื่อมต่อผ่าน **Bluetooth** หรือ **WiFi**

## 💳 การชำระเงิน

### Q: ลูกค้าจ่ายเงินสดไม่พอทอน?
**A:**
1. บอกลูกค้า**ยอดที่ขาด**
2. รับเงินเพิ่ม → ระบบคำนวณทอนใหม่
3. หรือ**ลดรายการ**บางอย่างออก

### Q: รับชำระบัตรเครดิตยังไงไง?
**A:**
- ต้องมี **เครื่อง EDC** เชื่อมต่อ
- เลือก **"บัตรเครดิต"** → ใส่ยอด → ให้ลูกค้าสวัดบัตร
- รอการอนุมัติ → พิมพ์ใบเสร็จ

### Q: QR Code ไม่แสดง?
**A:**
- ตรวจสอบ **อินเทอร์เน็ต**
- เช็ค **API Key** ของ Payment Gateway
- ลองกดปุ่ม **"สร้าง QR ใหม่"**

### Q: ลูกค้าจ่ายหลายช่องทาง (เงินสด + บัตร) ได้ไหม?
**A:**
- **ได้!** กดปุ่ม **"จ่ายแบบผสม"**
- ระบุยอดแต่ละช่องทาง
- ระบบจะคำนวณอัตโนมัติ

---

## 🎉 โปรโมชั่น

### Q: โปรโมชั่นไม่ทำงานอัตโนมัติ?
**A:** เช็ค:
- ✅ **โปรโมชั่นยังไม่หมดอายุ?**
- ✅ **สินค้าอยู่ในเงื่อนไขไหม?**
- ✅ **จำนวน/ยอดซื้อครบไหม?**
- ✅ **สาขาถูกต้องไหม?**

### Q: ใช้หลายโปรโมชั่นพร้อมกันได้ไหม?
**A:**
- **ได้** ถ้าโปรโมชั่นไม่ซ้ำซ้อนกัน
- ตัวอย่าง: ลดราคา + คูปอง + แลกแต้ม = ใช้พร้อมกันได้

### Q: คูปองใช้ไม่ได้?
**A:** เช็ค:
- **หมดอายุหรือยัง?**
- **ใช้ครบจำนวนแล้วหรือยัง?** (ถ้ามีจำกัด)
- **ยอดซื้อครบตามเงื่อนไขหรือยัง?**

### Q: สัญลักษณ์ ⏳ และ ✅ คืออะไร?
**A:**
- ⏳ = **กำลังจะได้** โปรโมชั่น (ยังไม่ครบเงื่อนไข)
- ✅ = **ได้แล้ว** (ใช้โปรโมชั่นแล้ว)

## 👥 สมาชิก

### Q: ลูกค้าไม่มีบัตรสมาชิก แต่ต้องการใช้แต้ม?
**A:**
- ให้ลูกค้า**กรอกเบอร์โทรศัพท์**
- ระบบค้นหาอัตโนมัติ
- หรือค้นหาจาก**ชื่อ-นามสกุล**

### Q: สแกนบาร์โค้ดบัตรสมาชิกไม่เจอ?
**A:**
- ลอง**พิมพ์เบอร์โทร**แทน
- หรือค้นหาจากชื่อ
- บาร์โค้ดอาจเสียหาย → พิมพ์บัตรใหม่

### Q: แต้มหายทำอย่างไร?
**A:**
- เช็ค **ประวัติการใช้แต้ม**
- ตรวจสอบว่า**หมดอายุหรือไม่**
- ติดต่อ Admin ตรวจสอบระบบ

### Q: เพิ่มสมาชิกได้กี่คน?
**A:**
- **ไม่จำกัด!** สมัครสมาชิกได้ตามต้องการ

## 🖨️ เครื่องพิมพ์

### Q: เครื่องพิมพ์ไม่ทำงาน?
**A:** วิธีแก้:
1. เช็ค**กระดาษพอไหม?**
2. **ปิด-เปิดเครื่อง**พิมพ์ใหม่
3. เช็ค**สายเชื่อมต่อ** (USB/LAN/Bluetooth)
4. ทดสอบ**พิมพ์ Test Page**

### Q: พิมพ์ออกมาตัวอักษรดำๆ/จางๆ?
**A:**
- **กระดาษไม่ถูกหน้า** → พลิกกระดาษ
- **หัวพิมพ์สกปรก** → เช็ดด้วยแอลกอฮอล์
- **กระดาษหมดอายุ** → เปลี่ยนม้วนใหม่

### Q: พิมพ์ใบเสร็จไม่ออก?
**A:**
- เช็คว่า**เลือกเครื่องพิมพ์ถูก**หรือไม่
- กดปุ่ม **"พิมพ์ซ้ำ"** ลองใหม่
- รีสตาร์ทโปรแกรม

### Q: ต้องการพิมพ์ใบเสร็จซ้ำ?
**A:**
- กดปุ่ม **"บิลเดิม"** หรือ **F6**
- เลือกบิลที่ต้องการ
- กด **"พิมพ์ซ้ำ"**

---

## ⚙️ ระบบและเทคนิค

### Q: โปรแกรมช้า/ค้าง?
**A:** วิธีแก้:
1. **ปิดโปรแกรมอื่นๆ** ที่ไม่จำเป็น
2. **ล้าง Cache:** เมนู → ตั้งค่า → ล้างข้อมูลชั่วคราว
3. **รีสตาร์ทคอมพิวเตอร์**
4. เช็ค**พื้นที่ Hard Disk**

### Q: ข้อมูลหาย/ไม่อัปเดต?
**A:**
- กดปุ่ม **"ซิงค์ข้อมูล"** (F11)
- ตรวจสอบ**อินเทอร์เน็ต**
- รอ 5-10 นาที แล้วลองใหม่

### Q: ขัดข้องขณะชำระเงิน?
**A:**
1. **อย่าปิดโปรแกรม!**
2. บิลจะถูกบันทึกเป็น**"บิลรอ"**อัตโนมัติ
3. เปิดโปรแกรมใหม่ → กด **"บิลรอ"**
4. เรียกบิลกลับมาชำระต่อ

### Q: ลืม Admin Password?
**A:**
- ติดต่อ **ฝ่ายสนับสนุน**: 02-XXX-XXXX
- จะต้องมีการ**ยืนยันตัวตน**
- ใช้ Master Key รีเซ็ต

## 🔧 การแก้ปัญหาเร่งด่วน

### 🚨 เครื่องค้างกะทันหัน

```
1. กด Ctrl+Alt+Del
2. เลือก Task Manager
3. หา BCaiPOS → End Task
4. เปิดโปรแกรมใหม่
5. กด "บิลรอ" เรียกบิลกลับมา
```

### 🚨 ไฟดับกลางคัน

```
1. รอไฟกลับมา
2. เปิดคอมพิวเตอร์และโปรแกรม
3. Login เข้าระบบ
4. กด "บิลรอ" → มีบิลที่ยังไม่ชำระ
5. เรียกบิลกลับมาชำระต่อ
```

### 🚨 อินเทอร์เน็ตขาด

```
✅ ขายได้ปกติ! (ใช้ข้อมูลออฟไลน์)
✅ บิลจะถูกบันทึกไว้
✅ เมื่อเน็ตกลับมา → ซิงค์อัตโนมัติ

⚠️ แต่:
❌ ไม่สามารถรับชำระ QR Code
❌ ไม่สามารถเช็คโปรโมชั่นใหม่
```

## 💡 เคล็ดลับเพิ่มเติม

### เพิ่มความเร็วการทำงาน

1. **ใช้ปุ่มลัด** (F1-F12)
2. **สแกนบาร์โค้ด** แทนการพิมพ์
3. **จำรหัสสินค้า** ที่ขายบ่อย
4. **เตรียมเงินทอน** ไว้ล่วงหน้า

### ลดข้อผิดพลาด

1. **ตรวจสอบ** ก่อนกด Enter
2. **ยืนยัน** กับลูกค้าก่อนชำระ
3. **นับเงิน** 2 ครั้ง (รับ-ทอน)
4. **เก็บใบเสร็จ** อย่างเป็นระเบียบ
