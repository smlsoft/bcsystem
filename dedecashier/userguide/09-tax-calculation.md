## ภาพรวมระบบภาษี

ระบบ DedeCashier รองรับการคำนวณภาษีมูลค่าเพิ่ม (VAT) แบบยืดหยุ่นตามมาตรฐานของประเทศไทย โดยสามารถตั้งค่าได้ที่ระดับ:

- **ระดับร้านค้า**: กำหนดอัตราภาษีเริ่มต้น (ปกติ 7%)
- **ระดับประเภทภาษี**: เลือกว่าราคาสินค้ารวมหรือแยกภาษี
- **ระดับสินค้า**: กำหนดว่าสินค้านั้นเสียภาษีหรือยกเว้นภาษี

### ค่าตัวแปรสำคัญ

```dart
// ค่าคงที่ในระบบ
vat_rate: double          // อัตราภาษี (เช่น 7.0 = 7%)
vat_type: int            // 0 = รวมใน, 1 = แยกนอก (ไม่ใช้ 2)
is_except_vat: bool      // true = ยกเว้นภาษี, false = เสียภาษี
bill_tax_type: int       // ประเภทเอกสารภาษี
is_vat_register: bool    // ร้านจดทะเบียนภาษีหรือไม่
```

## กฎหมายและข้อกำหนดที่เกี่ยวข้อง

### ฐานกฎหมาย
ระบบคำนวณภาษีใน DedeCashier เป็นไปตาม:
- **ประมวลรัษฎากร** มาตรา 77-91 (ภาษีมูลค่าเพิ่ม)
- **พระราชกำหนดภาษีมูลค่าเพิ่ม** พ.ศ. 2535 และที่แก้ไขเพิ่มเติม
- **กฎกระทรวงและประกาศอธิบดีกรมสรรพากร** ที่เกี่ยวข้อง

### อัตราภาษีมูลค่าเพิ่มในประเทศไทย

| ประเภท | อัตรา | คำอธิบาย |
|--------|-------|----------|
| **อัตราปกติ** | 7% | อัตราที่ใช้กับสินค้าและบริการทั่วไป (อาจเปลี่ยนแปลงตามกฎหมาย) |
| **อัตรา 0%** | 0% | ส่งออก, บริการที่ใช้ในต่างประเทศ (ขอคืนภาษีซื้อได้) |
| **ยกเว้นภาษี** | - | สินค้าเกษตร, หนังสือ, บริการบางประเภท (ขอคืนภาษีซื้อไม่ได้) |

### หน้าที่ผู้ประกอบการจดทะเบียน VAT

ระบบช่วยให้ผู้ประกอบการปฏิบัติตามหน้าที่ดังนี้:

#### 1. การออกใบกำกับภาษี
- ✅ ออกใบกำกับภาษีเต็มรูปสำหรับลูกค้าธุรกิจ
- ✅ ออกใบกำกับภาษีอย่างย่อสำหรับลูกค้ารายย่อย (ไม่เกิน 1,000 บาท)
- ✅ จัดเก็บสำเนาไว้อย่างน้อย 5 ปี

#### 2. การยื่นภาษี
- 📅 ยื่นแบบ ภ.พ.30 ภายในวันที่ 15 ของเดือนถัดไป
- 💰 ชำระภาษีภายในกำหนด (เลี่ยงเงินเพิ่ม 1.5% ต่อเดือน)
- 📊 จัดทำรายงานภาษีขาย-ซื้อ

#### 3. เกณฑ์การจดทะเบียน VAT
- รายรับ **เกิน 1.8 ล้านบาทต่อปี**: ต้องจดทะเบียน VAT
- รายรับ **ไม่เกิน 1.8 ล้านบาท**: สามารถเลือกจดหรือไม่จดก็ได้

## ประเภทภาษีที่รองรับ

### 1. ภาษีแบบรวมใน (VAT Included)
- `vat_type = 0`
- ราคาที่แสดงเป็นราคารวมภาษีแล้ว
- ต้องคำนวณย้อนกลับเพื่อหาราคาก่อนภาษี
- **ใช้กับ**: ร้านค้าปลีกทั่วไป, ซูเปอร์มาร์เก็ต

### 2. ภาษีแบบแยกนอก (VAT Excluded)
- `vat_type = 1`
- ราคาที่แสดงยังไม่รวมภาษี
- ภาษีจะถูกบวกเพิ่มเข้าไป
- **ใช้กับ**: ธุรกิจ B2B, โรงแรม, ร้านอาหารหรู

### 3. สินค้ายกเว้นภาษี (Non-VAT)
- `is_except_vat = true`
- ไม่คำนวณภาษีเลย
- **ใช้กับ**: สินค้าเกษตรสด, หนังสือ, สิ่งของบางประเภทตามกฎหมาย
- **หมายเหตุ**: ไม่สามารถนำภาษีซื้อมาขอคืนได้

### 4. ภาษีอัตรา 0% (Zero-Rated VAT)
- อัตราภาษี 0% แต่ยังนับเป็นธุรกรรมที่เสียภาษี
- **ใช้กับ**: สินค้าส่งออก, บริการที่ใช้ในต่างประเทศ
- **ข้อดี**: สามารถนำภาษีซื้อมาขอคืนได้
- **แตกต่างจากยกเว้นภาษี**: Zero-rated ขอคืนภาษีได้, ยกเว้นภาษีขอคืนไม่ได้

## สินค้าและบริการยกเว้นภาษี (VAT Exemption)

ตามมาตรา 81 แห่งประมวลรัษฎากร สินค้าและบริการที่ได้รับยกเว้น VAT:

### กลุ่มสินค้าเกษตร
- 🌾 พืชผลทางการเกษตร (ข้าว, ผัก, ผลไม้)
- 🐄 สัตว์มีชีวิตและซากสัตว์ (เนื้อสัตว์สด)
- 🌱 ปุ๋ย, ปลาป่น, อาหารสัตว์
- 💊 ยาและเคมีภัณฑ์สำหรับพืช/สัตว์

### กลุ่มสิ่งพิมพ์
- 📰 หนังสือพิมพ์
- 📖 นิตยสาร
- 📚 ตำราเรียน

### กลุ่มบริการพื้นฐาน
- 🚌 บริการขนส่งในประเทศ (ยกเว้นทางอากาศและท่อ)
- 🏠 บริการเช่าอสังหาริมทรัพย์

### กลุ่มบริการเพื่อสังคม
- 🏥 บริการรักษาพยาบาลของสถานพยาบาล
- 🎓 บริการด้านการศึกษา
- 📚 บริการห้องสมุด, พิพิธภัณฑ์, สวนสัตว์
- ⚖️ บริการประกอบโรคศิลปะ, สอบบัญชี, ว่าความ

### กลุ่มการเงิน
- 🏦 บริการของสถาบันการเงิน
- 💼 การประกันชีวิต

### กลุ่มรัฐบาล
- 🎫 สลากกินแบ่งรัฐบาล
- 📮 แสตมป์ไปรษณีย์ (ราคาไม่เกินมูลค่าที่ตรา)

## การคำนวณภาษีแบบรวมใน (VAT Included)

### หลักการ
เมื่อราคาสินค้ารวมภาษีแล้ว (107 บาท) ต้องคำนวณย้อนกลับเพื่อหา:
1. ราคาก่อนภาษี
2. จำนวนภาษี

### สูตรการคำนวณ

```
ราคาก่อนภาษี = ราคารวมภาษี / (1 + (อัตราภาษี / 100))
ภาษี = ราคารวมภาษี - ราคาก่อนภาษี
```

### ตัวอย่างโค้ด

```dart
// กรณี vat_type = 0 (รวมใน)
if (vat_type == 0 && !is_except_vat) {
  double priceExcludeVat = (price * 100) / (100 + vat_rate);
  double vatAmount = price - priceExcludeVat;
}
```

### ตัวอย่างการคำนวณ

#### ตัวอย่างที่ 1: สินค้าราคา 107 บาท (อัตรา VAT 7%)

```
ราคารวมภาษี = 107.00 บาท
อัตราภาษี = 7%

คำนวณ:
ราคาก่อนภาษี = 107 / (1 + 0.07)
               = 107 / 1.07
               = 100.00 บาท

ภาษี VAT = 107 - 100
          = 7.00 บาท

ตรวจสอบ: 100 + 7 = 107 ✅
```

#### ตัวอย่างที่ 2: สินค้าราคา 214 บาท (อัตรา VAT 7%)

```
ราคารวมภาษี = 214.00 บาท
อัตราภาษี = 7%

คำนวณ:
ราคาก่อนภาษี = 214 / 1.07
               = 200.00 บาท

ภาษี VAT = 214 - 200
          = 14.00 บาท
```

#### ตัวอย่างที่ 3: สินค้าราคา 53.50 บาท (อัตรา VAT 7%)

```
ราคารวมภาษี = 53.50 บาท
อัตราภาษี = 7%

คำนวณ:
ราคาก่อนภาษี = 53.50 / 1.07
               = 50.00 บาท

ภาษี VAT = 53.50 - 50.00
          = 3.50 บาท
```

### การคำนวณยอดรวมในบิล

```
สินค้า A: 107.00 บาท → ภาษี 7.00 บาท (ราคาก่อนภาษี 100 บาท)
สินค้า B: 214.00 บาท → ภาษี 14.00 บาท (ราคาก่อนภาษี 200 บาท)
สินค้า C: 53.50 บาท → ภาษี 3.50 บาท (ราคาก่อนภาษี 50 บาท)

รวมยอดรวมสินค้า:           374.50 บาท
รวมราคาก่อนภาษี:          350.00 บาท
รวมภาษี:                    24.50 บาท
────────────────────────────────────
ยอดรวมสุทธิ:              374.50 บาท
```

## การคำนวณภาษีแบบแยกนอก (VAT Excluded)

### หลักการ
เมื่อราคาสินค้ายังไม่รวมภาษี (100 บาท) ต้องคำนวณ:
1. ภาษี = ราคา × อัตราภาษี
2. ราคาสุทธิ = ราคา + ภาษี

### สูตรการคำนวณ

```
ภาษี = ราคาก่อนภาษี × (อัตราภาษี / 100)
ราคาสุทธิ = ราคาก่อนภาษี + ภาษี
```

### ตัวอย่างโค้ด

```dart
// กรณี vat_type = 1 (แยกนอก)
if (vat_type == 1 && !is_except_vat) {
  double priceExcludeVat = price;
  double vatAmount = price * (vat_rate / 100);
  double priceIncludeVat = price + vatAmount;
}
```

### ตัวอย่างการคำนวณ

#### ตัวอย่างที่ 1: สินค้าราคา 100 บาท (อัตรา VAT 7%)

```
ราคาก่อนภาษี = 100.00 บาท
อัตราภาษี = 7%

คำนวณ:
ภาษี VAT = 100 × 0.07
          = 7.00 บาท

ราคาสุทธิ = 100 + 7
           = 107.00 บาท
```

#### ตัวอย่างที่ 2: สินค้าราคา 200 บาท (อัตรา VAT 7%)

```
ราคาก่อนภาษี = 200.00 บาท
อัตราภาษี = 7%

คำนวณ:
ภาษี VAT = 200 × 0.07
          = 14.00 บาท

ราคาสุทธิ = 200 + 14
           = 214.00 บาท
```

#### ตัวอย่างที่ 3: สินค้าราคา 50 บาท (อัตรา VAT 7%)

```
ราคาก่อนภาษี = 50.00 บาท
อัตราภาษี = 7%

คำนวณ:
ภาษี VAT = 50 × 0.07
          = 3.50 บาท

ราคาสุทธิ = 50 + 3.50
           = 53.50 บาท
```

### การคำนวณยอดรวมในบิล

```
สินค้า A: 100.00 บาท + ภาษี 7.00 = 107.00 บาท
สินค้า B: 200.00 บาท + ภาษี 14.00 = 214.00 บาท
สินค้า C: 50.00 บาท + ภาษี 3.50 = 53.50 บาท

รวมราคาก่อนภาษี:          350.00 บาท
รวมภาษี:                    24.50 บาท
────────────────────────────────────
ยอดรวมสุทธิ:              374.50 บาท
```

## สินค้ายกเว้นภาษี (Non-VAT)

### หลักการ
สินค้าบางประเภทไม่ต้องเสียภาษีมูลค่าเพิ่มตามกฎหมาย

### เงื่อนไข

```dart
if (is_except_vat == true) {
  // ไม่คำนวณภาษี
  priceExcludeVat = price;
  vatAmount = 0;
  priceIncludeVat = price;
}
```

### ตัวอย่างการคำนวณ

```
สินค้าผักสด: 100.00 บาท (ยกเว้นภาษี)

ราคาก่อนภาษี = 100.00 บาท
ภาษี VAT = 0.00 บาท
ราคาสุทธิ = 100.00 บาท
```

### สินค้าที่ยกเว้นภาษีตามกฎหมาย (ตัวอย่าง)
- สินค้าเกษตรสด (ผัก, ผลไม้, เนื้อสัตว์สด)
- หนังสือ, หนังสือพิมพ์
- ยา, เวชภัณฑ์บางประเภท
- อาหารสัตว์
- ปุ๋ย, เมล็ดพันธุ์

## การคำนวณภาษีสำหรับตัวเลือกสินค้า (Options/Choices)

### หลักการ
เมื่อสินค้ามีตัวเลือก (เช่น เลือกท็อปปิ้ง, เลือกไซส์) ต้องคำนวณภาษีรวมกัน

### การคำนวณ

```dart
// 1. คำนวณราคาตัวเลือกรวมกับสินค้าหลัก
double totalPrice = detail.price + (sumamountchoice / detail.qty);

// 2. คำนวณภาษีตามประเภท
if (!detail.is_except_vat) {
  if (vat_type == 0) {
    // รวมใน
    priceExcludeVat = (totalPrice * 100) / (100 + vat_rate);
  } else {
    // แยกนอก
    priceExcludeVat = totalPrice;
  }
} else {
  // ยกเว้นภาษี
  priceExcludeVat = totalPrice;
}
```

### ตัวอย่างที่ 1: กาแฟ + ท็อปปิ้ง (แบบรวมภาษี)

```
กาแฟ:           50.00 บาท (รวมภาษีแล้ว)
+ วิปครีม:      10.00 บาท (รวมภาษีแล้ว)
+ ซอสคาราเมล:   5.00 บาท (รวมภาษีแล้ว)
────────────────────────────────
รวมราคา:        65.00 บาท

คำนวณภาษี (vat_type = 0, vat_rate = 7%):
ราคาก่อนภาษี = 65 / 1.07
               = 60.75 บาท

ภาษี VAT = 65 - 60.75
          = 4.25 บาท
```

### ตัวอย่างที่ 2: พิซซ่า + ท็อปปิ้ง (แบบแยกภาษี)

```
พิซซ่า:          200.00 บาท (ยังไม่รวมภาษี)
+ ชีสพิเศษ:      30.00 บาท (ยังไม่รวมภาษี)
+ เห็ดเพิ่ม:     20.00 บาท (ยังไม่รวมภาษี)
────────────────────────────────
รวมราคา:         250.00 บาท

คำนวณภาษี (vat_type = 1, vat_rate = 7%):
ราคาก่อนภาษี = 250.00 บาท

ภาษี VAT = 250 × 0.07
          = 17.50 บาท

ราคาสุทธิ = 250 + 17.50
           = 267.50 บาท
```

### ตัวอย่างที่ 3: ตัวเลือกราคา 0 บาท

```
กาแฟ:            50.00 บาท
+ น้ำตาล:        0.00 บาท (ไม่คิดเงิน)
+ นม:            0.00 บาท (ไม่คิดเงิน)
────────────────────────────────
รวมราคา:         50.00 บาท

ตัวเลือกราคา 0 บาทไม่มีผลต่อการคำนวณภาษี
```

## การคำนวณส่วนลด

### ประเภทส่วนลด

#### 1. ส่วนลดรายการ (Line Discount)
- ลดก่อนคำนวณภาษี
- ส่งผลต่อฐานภาษี

```dart
// ราคาหลังหักส่วนลด
double priceAfterDiscount = price - discount;

// จากนั้นคำนวณภาษีจากราคาหลังหักส่วนลด
```

## สูตรคำนวณทั้งหมด

### สูตรพื้นฐาน

#### 1. แบบรวมภาษี (VAT Included)
```
ราคาก่อนภาษี = ราคารวมภาษี / (1 + (อัตราภาษี / 100))
ภาษี = ราคารวมภาษี - ราคาก่อนภาษี

หรือ (สูตรเดียว):
ภาษี = ราคารวมภาษี × (อัตราภาษี / (100 + อัตราภาษี))
```

#### 2. แบบแยกภาษี (VAT Excluded)
```
ภาษี = ราคาก่อนภาษี × (อัตราภาษี / 100)
ราคาสุทธิ = ราคาก่อนภาษี + ภาษี
```

#### 3. ยกเว้นภาษี
```
ราคาก่อนภาษี = ราคา
ภาษี = 0
ราคาสุทธิ = ราคา
```

### สูตรคำนวณบิลรวม

```
สำหรับสินค้าแต่ละรายการ:

1. คำนวณราคาหลังส่วนลดรายการ:
   ราคาสุทธิรายการ = (ราคา × จำนวน) - ส่วนลด

2. คำนวณภาษีตามประเภท (ใช้สูตรข้างต้น)

3. รวมทุกรายการ:
   รวมราคาก่อนภาษี = Σ(ราคาก่อนภาษีแต่ละรายการ)
   รวมภาษี = Σ(ภาษีแต่ละรายการ)
   รวมยอดรวม = Σ(ราคาสุทธิแต่ละรายการ)

4. หักส่วนลดท้ายบิล (ถ้ามี):
   ยอดชำระ = รวมยอดรวม - ส่วนลดท้ายบิล

หมายเหตุ: ภาษียังคงเท่าเดิม ไม่เปลี่ยนแปลงจากส่วนลดท้ายบิล
```

### โค้ดอ้างอิง

```dart
// ตัวอย่างโค้ดจริงในระบบ (sync_bill.dart)

double priceExcludeVat = 0;

if (!detail.is_except_vat) {
  if (bill.vat_type == 0) {
    // ภาษีรวมใน
    priceExcludeVat = ((detail.price + (sumamountchoice / detail.qty)) * 100) 
                      / (100 + bill.vat_rate);
  } else {
    // ภาษีแยกนอก
    priceExcludeVat = detail.price + (sumamountchoice / detail.qty);
  }
} else {
  // ยกเว้นภาษี
  priceExcludeVat = detail.price + (sumamountchoice / detail.qty);
}

// คำนวณภาษี
double totalValueVat = ((detail.price * detail.qty) + sumamountchoice) 
                       - (priceExcludeVat * detail.qty);
```

## จุดความรับผิดในการเสียภาษี (Tax Point)

### การขายสินค้า (มาตรา 78(1))

ความรับผิดในการเสียภาษีเกิดขึ้นเมื่อ **ส่งมอบสินค้า** เป็นหลัก

**ยกเว้น** กรณีที่เหตุการณ์ใดเหตุการณ์หนึ่งเกิดขึ้นก่อน:

| เหตุการณ์ | คำอธิบาย | Tax Point |
|-----------|----------|-----------|
| **โอนกรรมสิทธิ์** | จดทะเบียนโอนรถ, โอนที่ดิน | เมื่อโอนกรรมสิทธิ์ |
| **รับชำระเงิน** | ได้รับเงินก่อนส่งของ (มัดจำ, เงินล่วงหน้า) | เมื่อได้รับเงิน |
| **ออกใบกำกับภาษี** | ออกใบกำกับภาษีก่อนส่งของ | เมื่อออกใบกำกับภาษี |

#### ตัวอย่าง: Tax Point การขายสินค้า

```
สถานการณ์ที่ 1: ลูกค้ามาซื้อสินค้าและชำระเงินพร้อมรับสินค้าทันที
→ Tax Point = วันที่ส่งมอบสินค้า

สถานการณ์ที่ 2: ลูกค้าจ่ายเงินมัดจำ 50% วันที่ 1 ม.ค., ส่งสินค้าวันที่ 15 ม.ค.
→ Tax Point แรก = 1 ม.ค. (50% ของภาษี)
→ Tax Point ที่สอง = 15 ม.ค. (50% ที่เหลือ)

สถานการณ์ที่ 3: ออกใบกำกับภาษีวันที่ 5 ม.ค., ส่งสินค้าวันที่ 10 ม.ค.
→ Tax Point = 5 ม.ค. (วันที่ออกใบกำกับภาษี)
```

### การให้บริการ (มาตรา 78/3)

ความรับผิดในการเสียภาษีเกิดขึ้นเมื่อ **ชำระราคาค่าบริการ** เป็นหลัก

**ยกเว้น** กรณีที่เหตุการณ์ใดเหตุการณ์หนึ่งเกิดขึ้นก่อน:

| เหตุการณ์ | คำอธิบาย | Tax Point |
|-----------|----------|-----------|
| **ได้ใช้บริการ** | ใช้บริการแล้วตามส่วนที่ใช้ | เมื่อใช้บริการ |
| **ออกใบกำกับภาษี** | ออกใบกำกับภาษีก่อนชำระเงิน | เมื่อออกใบกำกับภาษี |

#### ตัวอย่าง: Tax Point การให้บริการ

```
สถานการณ์ที่ 1: ร้านอาหาร - ลูกค้ารับประทานอาหารและชำระเงินทันที
→ Tax Point = วันที่ชำระเงิน

สถานการณ์ที่ 2: โรงแรม - เข้าพักวันที่ 1-5 ม.ค., ชำระเงินวันที่ 5 ม.ค.
→ Tax Point = 5 ม.ค. (วันที่ชำระเงิน)

สถานการณ์ที่ 3: ให้บริการรายเดือน - ออกใบแจ้งหนี้ล่วงหน้าวันที่ 1
→ Tax Point = วันที่ 1 (วันที่ออกใบกำกับภาษี)
```

### กรณีพิเศษ: ร้านค้าปลีก

สำหรับร้านค้าปลีกที่ขาย/ให้บริการมูลค่าไม่เกิน **1,000 บาทต่อครั้ง**:
- ✅ สามารถจัดทำใบกำกับภาษีอย่างย่อรวมรายวันได้
- ✅ ไม่ต้องออกใบกำกับภาษีทันที (ยกเว้นลูกค้าร้องขอ)
- ✅ Tax Point = วันทำการนั้น

## การเฉลี่ยภาษีขายกรณีมีส่วนลดท้ายบิล

### หลักการสำคัญ

เมื่อมีการขายสินค้าหลายประเภทในบิลเดียวกัน โดยมีทั้ง:
- สินค้าที่เสีย VAT 7%
- สินค้ายกเว้น VAT
- มีส่วนลดท้ายบิล

ต้อง **ปันส่วนส่วนลด** ตามสัดส่วนมูลค่าสินค้าแต่ละประเภท

### ขั้นตอนการคำนวณ

#### ตัวอย่าง: บิลผสมมีส่วนลดท้ายบิล

```
รายการสินค้า:
1. เสื้อผ้า (เสีย VAT):     500.00 บาท
2. ผักสด (ยกเว้น VAT):     300.00 บาท
                           ─────────────
รวม:                        800.00 บาท
ส่วนลดท้ายบิล:              -50.00 บาท
                           ─────────────
ยอดหลังส่วนลด:             750.00 บาท
```

**ขั้นตอนที่ 1: คำนวณสัดส่วน**

```
สัดส่วนเสื้อผ้า = 500 / 800 = 0.625 (62.5%)
สัดส่วนผักสด = 300 / 800 = 0.375 (37.5%)
```

**ขั้นตอนที่ 2: ปันส่วนส่วนลด**

```
ส่วนลดเสื้อผ้า = 50 × 0.625 = 31.25 บาท
ส่วนลดผักสด = 50 × 0.375 = 18.75 บาท
```

**ขั้นตอนที่ 3: คำนวณฐานภาษี**

```
เสื้อผ้า (เสีย VAT):
  ฐานภาษี = 500 - 31.25 = 468.75 บาท
  VAT 7% = 468.75 × 0.07 = 32.81 บาท
  ราคารวม VAT = 468.75 + 32.81 = 501.56 บาท

ผักสด (ยกเว้น VAT):
  ราคาสุทธิ = 300 - 18.75 = 281.25 บาท
  VAT = 0.00 บาท
```

**สรุป:**

```
รายการ              ฐานภาษี     VAT      ราคารวม
────────────────────────────────────────────────
เสื้อผ้า            468.75    32.81     501.56
ผักสด               281.25     0.00     281.25
────────────────────────────────────────────────
รวม                 750.00    32.81     782.81
```

⚠️ **การเฉลี่ยภาษีซื้อ (Input Tax Apportionment)**

หากธุรกิจมีทั้งรายได้เสีย VAT และยกเว้น VAT:
- ต้องเฉลี่ยภาษีซื้อสำหรับค่าใช้จ่ายส่วนกลาง
- เช่น ค่าเช่า, ค่าน้ำ-ไฟ, ค่าโฆษณา
- นำภาษีซื้อไปหักได้เฉพาะสัดส่วนรายได้ที่เสีย VAT

**สูตรการเฉลี่ย:**
```
ภาษีซื้อที่หักได้ = ภาษีซื้อทั้งหมด × (รายได้เสีย VAT / รายได้ทั้งหมด)
```

---

## การตรวจสอบความถูกต้อง

### วิธีตรวจสอบ

#### 1. แบบรวมภาษี
```
ราคาก่อนภาษี + ภาษี = ราคารวมภาษี ✅

ตัวอย่าง:
100 + 7 = 107 ✅
```

#### 2. แบบแยกภาษี
```
ราคาก่อนภาษี × (1 + อัตราภาษี/100) = ราคาสุทธิ ✅

ตัวอย่าง:
100 × 1.07 = 107 ✅
```

#### 3. บิลรวม
```
ผลรวมทุกรายการ = ผลรวมราคาก่อนภาษี + ผลรวมภาษี ✅

ตัวอย่าง:
(100 + 200 + 50) + (7 + 14 + 3.50) = 374.50 ✅
```

## คำถามที่พบบ่อย (FAQ)

### Q1: ทำไมต้องมีทั้งแบบรวมและแยกภาษี?
**A:** ขึ้นอยู่กับประเภธุรกิจ:
- **ร้านค้าปลีก**: ใช้แบบรวมภาษี (ราคาที่เห็นคือราคาจ่าย)
- **B2B/โรงแรม**: ใช้แบบแยกภาษี (บวก VAT ตอนจ่าย)

### Q2: สินค้าอะไรบ้างที่ยกเว้นภาษี?
**A:** ตามกฎหมายของประเทศไทย เช่น:
- สินค้าเกษตรสด
- หนังสือ หนังสือพิมพ์
- ยา เวชภัณฑ์บางประเภท

### Q3: ส่วนลดมีผลต่อภาษีอย่างไร?
**A:**
- **ส่วนลดรายการ**: ลดก่อนคำนวณภาษี → ภาษีลดด้วย
- **ส่วนลดท้ายบิล**: ลดหลังคำนวณภาษี → ภาษีไม่เปลี่ยน

### Q4: อัตราภาษีสามารถเปลี่ยนได้หรือไม่?
**A:** ได้ ระบบรองรับการตั้งค่าอัตราภาษี (ปกติคือ 7% แต่สามารถปรับได้)

### Q5: ถ้าสินค้ามีตัวเลือก ภาษีคำนวณยังไง?
**A:** รวมราคาสินค้าหลัก + ตัวเลือกทั้งหมดก่อน จึงคำนวณภาษี

### Q6: Service Charge คิดยังไง?
**A:** Service Charge มักคิดจากยอดหลังรวมภาษีแล้ว และไม่เสีย VAT

### Q7: ถ้าขายทั้งสินค้าเสีย VAT และยกเว้น VAT พร้อมส่วนลดท้ายบิล ต้องคำนวณยังไง?
**A:** ต้องปันส่วนส่วนลดตามสัดส่วนมูลค่าสินค้าแต่ละประเภทก่อนคำนวณภาษี (ดูส่วนการเฉลี่ยภาษีขาย)

### Q8: Tax Point คืออะไร?
**A:** จุดเวลาที่เกิดความรับผิดในการเสียภาษี:
- **การขายสินค้า**: เมื่อส่งมอบสินค้า (หรือเมื่อโอนกรรมสิทธิ์/รับเงิน/ออกใบกำกับภาษีก่อน)
- **การให้บริการ**: เมื่อชำระเงิน (หรือเมื่อใช้บริการ/ออกใบกำกับภาษีก่อน)

### Q9: ร้านค้าปลีกต้องออกใบกำกับภาษีทุกครั้งหรือไม่?
**A:** 
- มูลค่าไม่เกิน 1,000 บาท: สามารถออกใบกำกับภาษีอย่างย่อรวมรายวันได้
- มูลค่าเกิน 1,000 บาท หรือลูกค้าร้องขอ: ต้องออกใบกำกับภาษีเต็มรูป

### Q10: ธุรกิจรายรับน้อยต้องจดทะเบียน VAT หรือไม่?
**A:** 
- รายรับเกิน 1.8 ล้านบาท/ปี: **ต้องจด** ทะเบียน VAT
- รายรับไม่เกิน 1.8 ล้านบาท/ปี: **เลือกจดหรือไม่จดก็ได้**

### Q11: ถ้าไม่ชำระภาษีตรงเวลาจะเป็นอย่างไร?
**A:** ต้องเสียเงินเพิ่ม 1.5% ต่อเดือน ของภาษีที่ต้องชำระ

### Q12: ภาษีอัตรา 0% กับยกเว้นภาษีต่างกันอย่างไร?
**A:**
- **ภาษี 0%**: ขอคืนภาษีซื้อได้ (เช่น ส่งออก)
- **ยกเว้นภาษี**: ขอคืนภาษีซื้อไม่ได้ (เช่น สินค้าเกษตร)

## สรุปสูตรคำนวณภาษีทั้งหมด

### สูตรหลัก

```
1. ภาษีแบบรวมใน (VAT Included):
   ราคาก่อนภาษี = ราคารวมภาษี / (1 + อัตราภาษี/100)
   ภาษี = ราคารวมภาษี - ราคาก่อนภาษี

2. ภาษีแบบแยกนอก (VAT Excluded):
   ภาษี = ราคาก่อนภาษี × (อัตราภาษี/100)
   ราคาสุทธิ = ราคาก่อนภาษี + ภาษี

3. ยกเว้นภาษี (Non-VAT):
   ราคาก่อนภาษี = ราคา
   ภาษี = 0
   ราคาสุทธิ = ราคา

4. การปันส่วนส่วนลดท้ายบิล:
   สัดส่วนสินค้า = มูลค่าสินค้า / มูลค่ารวมทั้งหมด
   ส่วนลดที่ปันส่วน = ส่วนลดท้ายบิล × สัดส่วนสินค้า
   ฐานภาษี = มูลค่าสินค้า - ส่วนลดที่ปันส่วน

5. การเฉลี่ยภาษีซื้อ:
   ภาษีซื้อที่หักได้ = ภาษีซื้อทั้งหมด × (รายได้เสีย VAT / รายได้ทั้งหมด)
```

## ภาคผนวก: ตัวอย่างโค้ดจริงในระบบ

### การคำนวณภาษีสินค้าหลัก

```dart
// จาก lib/api/sync/sync_bill.dart (บรรทัด 163-192)

double priceExcludeVat = 0;

if (!detail.is_except_vat) {
  if (bill.vat_type == 0) {
    // ภาษีรวมใน (VAT Included)
    priceExcludeVat = ((detail.price + (sumamountchoice / detail.qty)) * 100) 
                      / (100 + bill.vat_rate);
  } else {
    // ภาษีแยกนอก (VAT Excluded)
    priceExcludeVat = detail.price + (sumamountchoice / detail.qty);
  }
} else {
  // ยกเว้นภาษี (Non-VAT)
  priceExcludeVat = detail.price + (sumamountchoice / detail.qty);
}

// คำนวณยอด VAT
double totalValueVat = ((detail.price * detail.qty) + sumamountchoice) 
                       - (priceExcludeVat * detail.qty);
```

### การคำนวณภาษีตัวเลือกสินค้า (Options)

```dart
// จาก lib/api/sync/sync_bill.dart (บรรทัด 248-273)

double priceChoiceExcludeVat = 0;

if (!detail.is_except_vat) {
  if (global.posConfig.vattype == 0) {
    // ภาษีรวมใน - คำนวณย้อนกลับ
    priceChoiceExcludeVat = (option.price * 100) / (100 + global.posConfig.vatrate);
  } else {
    // ภาษีแยกนอก - ราคาเท่าเดิม
    priceChoiceExcludeVat = option.price;
  }
} else {
  // ยกเว้นภาษี
  priceChoiceExcludeVat = option.price;
}

// คำนวณยอด VAT ของตัวเลือก
double totalValueVat = (option.price * detail.qty) - (priceChoiceExcludeVat * detail.qty);
```

### ข้อมูลบิลในโมเดล

```dart
// จาก lib/model/objectbox/bill_struct.dart

class BillObjectBoxStruct {
  double vat_rate;              // อัตราภาษี (7.0 = 7%)
  int vat_type;                // 0 = รวมใน, 1 = แยกนอก
  bool is_vat_register;        // ร้านจดทะเบียน VAT หรือไม่
  
  double amount_before_calc_vat;  // มูลค่าก่อนคำนวณภาษี
  double amount_after_calc_vat;   // มูลค่าหลังคำนวณภาษี
  double amount_except_vat;       // มูลค่าสินค้ายกเว้นภาษี
  double total_vat_amount;        // ภาษี VAT รวม
  double total_amount;            // ยอดรวมทั้งสิ้น
}
```
