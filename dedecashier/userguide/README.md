## 🎯 วัตถุประสงค์ของคู่มือนี้

คู่มือนี้จัดทำขึ้นเพื่อ:
- ✅ **สอนพนักงานใหม่** ให้ใช้งานระบบได้อย่างรวดเร็ว
- ✅ **อ้างอิงเมื่อมีปัญหา** ค้นหาวิธีแก้ได้ง่าย
- ✅ **ลดเวลาการฝึกอบรม** เรียนรู้ด้วยตนเองได้
- ✅ **เพิ่มประสิทธิภาพการทำงาน** ใช้ปุ่มลัดและเทคนิคต่างๆ

## 🔄 การอัปเดต

คู่มือนี้จะมีการอัปเดตเป็นประจำเมื่อมีฟีเจอร์ใหม่

**อัปเดตล่าสุด:**
- ✨ เพิ่มการสแกนบาร์โค้ดแบบ quantity*barcode (เช่น 10*8850123456789)
- ✨ เพิ่มการแสดง overlay แบบ real-time
- ✨ ปรับปรุงการค้นหาสินค้าและลูกค้า
