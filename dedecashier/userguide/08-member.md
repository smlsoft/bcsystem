## 🎯 ภาพรวมระบบสมาชิก

### ประโยชน์ของระบบสมาชิก

| สำหรับร้านค้า | สำหรับลูกค้า |
|--------------|-------------|
| 📊 เก็บข้อมูลลูกค้า | ⭐ สะสมแต้มทุกครั้งที่ซื้อ |
| 💰 เพิ่มยอดขาย | 🎁 แลกของรางวัล |
| 🎯 วิเคราะห์พฤติกรรม | 💸 รับส่วนลดพิเศษ |
| 📱 ส่งโปรโมชั่น | 🎂 โปรโมชั่นวันเกิด |

## 🔍 การค้นหาสมาชิก

### วิธีที่ 1: กดปุ่มค้นหาสมาชิก

1. กดปุ่ม **👤 สมาชิก** ที่ด้านล่างหน้าจอ
2. หน้าจอจะเปลี่ยนเป็นโหมดค้นหาสมาชิก
3. พิมพ์ข้อมูลใดข้อมูลหนึ่ง:
   - 📱 เบอร์โทรศัพท์
   - 🏷️ รหัสสมาชิก
   - 👤 ชื่อ-นามสกุล
4. กด **Enter** หรือ **คลิกผลลัพธ์**
5. ระบบจะแสดงข้อมูลสมาชิกทันที

### วิธีที่ 2: สแกนบัตรสมาชิก

1. ใช้เครื่องสแกนบาร์โค้ด
2. **สแกนบาร์โค้ดบนบัตรสมาชิก**
3. ระบบจะค้นหาและแสดงข้อมูลอัตโนมัติ

### วิธีที่ 3: พิมพ์เบอร์โทรศัพท์

1. อยู่ที่หน้าจอหลัก (ปุ่ม #)
2. พิมพ์**เบอร์โทรศัพท์** 10 หลัก
3. กด **Enter**
4. ระบบจะค้นหาสมาชิกอัตโนมัติ

## 🆕 การเพิ่มสมาชิกใหม่

### 📝 ขั้นตอนการสมัครสมาชิก

#### 1. เปิดฟอร์มสมัครสมาชิก

```
กดปุ่ม "เพิ่มสมาชิก" หรือ F10
```

#### 2. กรอกข้อมูลที่จำเป็น

| ฟิลด์ | จำเป็น | หมายเหตุ |
|------|-------|----------|
| **ชื่อ** | ✅ | ชื่อจริง |
| **นามสกุล** | ✅ | นามสกุล |
| **เบอร์โทร** | ✅ | เบอร์มือถือ 10 หลัก |
| **อีเมล** | ❌ | สำหรับส่งโปรโมชั่น |
| **วันเกิด** | ❌ | รับโปรโมชั่นวันเกิด |
| **ที่อยู่** | ❌ | สำหรับจัดส่งสินค้า |
| **เพศ** | ❌ | ชาย/หญิง/ไม่ระบุ |

#### 3. เลือกประเภทสมาชิก

```
[1] ทั่วไป      - ไม่มีค่าสมัคร
[2] เงิน        - ฝากเงินไว้ใช้
[3] VIP         - สะสมแต้ม 2 เท่า
```

#### 4. บันทึกข้อมูล

- กดปุ่ม **"บันทึก"** หรือ **Enter**
- ระบบจะสร้างรหัสสมาชิกอัตโนมัติ
- พิมพ์บัตรสมาชิก (ถ้ามี)

## ⭐ การสะสมและใช้แต้ม

### 💯 การสะสมแต้ม

#### กฎการสะสมแต้ม (ปรับได้ตามร้าน)

```
ซื้อ 100 บาท = 10 แต้ม
ซื้อ 1,000 บาท = 100 แต้ม

สมาชิก VIP = แต้ม 2 เท่า
วันเกิด = แต้ม 3 เท่า
```

#### ตัวอย่างการสะสมแต้ม

```
┌─────────────────────────────────┐
│  👤 คุณสมชาย ใจดี                │
│  📞 081-234-5678                 │
├─────────────────────────────────┤
│                                 │
│  ยอดซื้อ:          1,500 บาท    │
│  อัตราแต้ม:        10:1          │
│  แต้มที่ได้:       +150 แต้ม    │
│                                 │
│  💰 แต้มก่อนหน้า:  1,200 แต้ม   │
│  ⭐ แต้มหลังซื้อ:  1,350 แต้ม   │
│                                 │
└─────────────────────────────────┘
```

### 🎁 การใช้แต้ม

#### วิธีที่ 1: แลกของรางวัล

**ขั้นตอน:**

1. **ระบุสมาชิก** (สแกนบัตร/กรอกเบอร์)
2. กดปุ่ม **"แลกแต้ม"** หรือ **F8**
3. **เลือกของแลก** จากรายการ
4. ยืนยันการแลก
5. ระบบหักแต้มอัตโนมัติ

**ตัวอย่างของแลก:**

```
┌─────────────────────────────────┐
│  🎁 ของรางวัล                    │
├─────────────────────────────────┤
│                                 │
│  [1] แก้วน้ำ       500 แต้ม     │
│  [2] ร่ม           800 แต้ม     │
│  [3] เสื้อยืด     1,200 แต้ม    │
│  [4] กระเป๋า      1,500 แต้ม    │
│  [5] รองเท้า      2,000 แต้ม    │
│                                 │
│  💯 แต้มคงเหลือ: 1,350 แต้ม     │
└─────────────────────────────────┘
```

#### วิธีที่ 2: แลกส่วนลด

**ขั้นตอน:**

1. **ระบุสมาชิก**
2. กดปุ่ม **"ใช้แต้มลด"**
3. **ระบุจำนวนแต้ม** ที่ต้องการใช้
4. ระบบคำนวณส่วนลด (เช่น 100 แต้ม = 10 บาท)
5. แสดงยอดชำระหลังหักส่วนลด

**ตัวอย่าง:**

```
ยอดซื้อรวม:         1,000 บาท

💯 แต้มคงเหลือ:     1,350 แต้ม
ใช้แต้ม:            -500 แต้ม
ส่วนลด:              -50 บาท

────────────────────────────────
ยอดชำระ:             950 บาท
แต้มคงเหลือ:         850 แต้ม
แต้มที่ได้ครั้งนี้:  +95 แต้ม
```

## ✏️ การแก้ไขข้อมูลสมาชิก

### 🔧 ขั้นตอนการแก้ไข

1. **ค้นหาสมาชิก** ที่ต้องการแก้ไข
2. กดปุ่ม **"แก้ไข"** หรือ **F4**
3. **แก้ไขข้อมูล** ที่ต้องการ
4. กดปุ่ม **"บันทึก"**
5. ระบบจะอัปเดตข้อมูลทันที

### 📝 ข้อมูลที่สามารถแก้ไขได้

| ข้อมูล | แก้ไขได้ |
|--------|---------|
| ชื่อ-นามสกุล | ✅ |
| เบอร์โทรศัพท์ | ✅ |
| อีเมล | ✅ |
| วันเกิด | ✅ |
| ที่อยู่ | ✅ |
| ระดับสมาชิก | ✅ (Admin เท่านั้น) |
| รหัสสมาชิก | ❌ |
| วันที่สมัคร | ❌ |

## 🏆 ระดับสมาชิก

### ประเภทสมาชิก

#### 1. 🥉 สมาชิกทั่วไป (General)

```
✓ สะสมแต้ม 1 เท่า
✓ แลกของรางวัลทั่วไป
✓ รับโปรโมชั่นทั่วไป
✓ ไม่มีค่าสมัคร
```

#### 2. 🥈 สมาชิกเงิน (Deposit)

```
✓ ฝากเงินไว้ใช้ล่วงหน้า
✓ ซื้อสินค้าหักจากยอดฝาก
✓ ไม่ต้องพกเงินสด
✓ เติมเงินได้ตลอด
```

#### 3. 🥇 สมาชิก VIP

```
✓ สะสมแต้ม 2 เท่า
✓ ส่วนลดพิเศษทุกรายการ
✓ แลกของรางวัลพิเศษ
✓ แถวด่วน VIP (ถ้ามี)
✓ โปรโมชั่นสำหรับ VIP
```

### 📊 การเลื่อนระดับสมาชิก

#### เงื่อนไขการเลื่อนระดับ (ตัวอย่าง)

```
ทั่วไป → VIP
• ซื้อสะสม 50,000 บาท
• หรือ มียอดซื้อเฉลี่ย 5,000 บาท/เดือน
• เป็นสมาชิกมากกว่า 6 เดือน
```

#### สิทธิพิเศษเมื่อเลื่อนระดับ

```
🎉 โบนัสแต้ม 1,000 คะแนน
🎁 ของขวัญพิเศษ
💳 บัตร VIP Card
📱 เข้ากลุ่ม LINE VIP
```

## 💳 สมาชิกประเภทเงินฝาก

### การเติมเงิน

**ขั้นตอน:**

1. **ค้นหาสมาชิก**
2. กดปุ่ม **"เติมเงิน"**
3. **ระบุจำนวนเงิน** ที่ต้องการเติม
4. **รับเงินจากลูกค้า**
5. บันทึก
6. พิมพ์ใบเสร็จเติมเงิน

**ตัวอย่างใบเสร็จเติมเงิน:**

```
================================
      ใบเสร็จเติมเงิน
================================

สมาชิก: คุณสมชาย ใจดี
รหัส: M000123
เบอร์: 081-234-5678

ยอดเดิม:           5,000 บาท
เติมเงิน:         +2,000 บาท
────────────────────────────
ยอดคงเหลือ:        7,000 บาท

วันที่: 15/10/2025 14:30
================================
```

---

### การใช้เงินฝาก

**ขั้นตอนชำระด้วยเงินฝาก:**

1. **ระบุสมาชิก** ก่อนสแกนสินค้า
2. สแกนสินค้าตามปกติ
3. เมื่อชำระเงิน เลือก **"หักจากเงินฝาก"**
4. ระบบหักอัตโนมัติ
5. พิมพ์ใบเสร็จ (แสดงยอดเงินคงเหลือ)

**ตัวอย่าง:**

```
ยอดซื้อ:            1,500 บาท

💰 ยอดเงินฝากเดิม:   7,000 บาท
   หักจากเงินฝาก:  -1,500 บาท
────────────────────────────
💵 เงินฝากคงเหลือ:   5,500 บาท
```

## ⚠️ ข้อควรระวัง

### ❌ ข้อห้าม

- **ห้าม** แก้ไขแต้มโดยไม่ได้รับอนุญาต
- **ห้าม** ลบข้อมูลสมาชิกโดยพลการ
- **ห้าม** เปิดเผยข้อมูลส่วนตัวลูกค้า
- **ห้าม** ให้คนอื่นใช้บัตรสมาชิกแทน

### ✅ ข้อควรทำ

- **ตรวจสอบ** ตัวตนก่อนให้ใช้แต้ม
- **อธิบาย** กฎการสะสมแต้มให้ลูกค้าเข้าใจ
- **แจ้งยอดแต้ม** ให้ลูกค้าทราบทุกครั้ง
- **แนะนำ** ของรางวัลที่แลกได้

## ❓ คำถามที่พบบ่อย

**Q: ลูกค้าไม่มีบัตรสมาชิก จะใช้แต้มได้ไหม?**  
A: ได้! ให้ลูกค้ากรอกเบอร์โทรศัพท์แทน

**Q: สแกนบัตรสมาชิกแล้วไม่เจอ?**  
A: ตรวจสอบว่าบาร์โค้ดชัดเจน หรือค้นหาด้วยเบอร์โทร

**Q: แต้มหมดอายุไหม?**  
A: ขึ้นอยู่กับนโยบายร้าน (โดยทั่วไป 1-2 ปี)

**Q: แลกแต้มแล้วคืนสินค้าได้ไหม?**  
A: ได้ แต้มจะคืนอัตโนมัติ

**Q: สมาชิกลืมบัตรทำอย่างไร?**  
A: ค้นหาด้วยเบอร์โทรศัพท์
