# เริ่มต้นใช้งาน BC Ai POS

## 📋 สิ่งที่ต้องเตรียม

ก่อนเริ่มใช้งาน คุณต้องได้รับจากผู้ดูแลระบบ:
- ✅ **ชื่อผู้ใช้** (Username)
- ✅ **รหัสผ่าน** (Password)
- ✅ **ชื่อร้าน** (ถ้ามีหลายสาขา)
- ✅ **ลิงก์ดาวน์โหลดโปรแกรม**

## 💾 การติดตั้ง

### สำหรับ Windows

#### ขั้นตอนที่ 1: ดาวน์โหลด
1. เปิดลิงก์ที่ได้รับจากผู้ดูแล
2. ดาวน์โหลดไฟล์ติดตั้ง (`.exe` หรือ `.msix`)
3. บันทึกไฟล์ไว้บนเดสก์ท็อป

#### ขั้นตอนที่ 2: ติดตั้ง
1. ดับเบิลคลิกไฟล์ที่ดาวน์โหลด
2. หาก Windows แจ้งเตือน "Windows protected your PC"
   - คลิก "More info"
   - คลิก "Run anyway"
3. คลิก "ถัดไป" → "ติดตั้ง" → "เสร็จสิ้น"
4. โปรแกรมจะติดตั้งอัตโนมัติ (ใช้เวลา 2-3 นาที)

#### ขั้นตอนที่ 3: เปิดโปรแกรม
1. คลิกไอคอน "BC Ai POS" บนเดสก์ท็อป
2. หรือค้นหาใน Start Menu
3. รอโปรแกรมเปิด (ครั้งแรกอาจใช้เวลา 10-20 วินาที)

### สำหรับ Android Tablet

#### วิธีที่ 1: ผ่าน Google Play Store
1. เปิด Play Store
2. ค้นหา "BC Ai POS"
3. กดปุ่ม "ติดตั้ง"
4. รอจนติดตั้งเสร็จ
5. กด "เปิด"

#### วิธีที่ 2: ผ่านไฟล์ APK
1. ดาวน์โหลดไฟล์ `.apk` จากลิงก์ที่ได้รับ
2. เปิดการตั้งค่า → ความปลอดภัย
3. เปิด "ติดตั้งจากแหล่งที่ไม่รู้จัก"
4. เปิดไฟล์ APK และกด "ติดตั้ง"
5. กด "เปิด" เมื่อติดตั้งเสร็จ

## 🔌 เชื่อมต่ออุปกรณ์

### 1. เครื่องสแกนบาร์โค้ด

#### แบบ USB
1. เสียบสาย USB เข้ากับคอมพิวเตอร์
2. รอ Windows ติดตั้งไดร์เวอร์อัตโนมัติ (10-30 วินาที)
3. เมื่อมีเสียง "ดิ๊ง" แสดงว่าพร้อมใช้งาน
4. **ทดสอบ:** สแกนบาร์โค้ดสินค้าดู

#### แบบ Bluetooth
1. เปิดเครื่องสแกน (กดปุ่ม Power ค้างไว้ 2-3 วินาที)
2. เปิด Bluetooth บนคอมพิวเตอร์
3. ค้นหาอุปกรณ์ Bluetooth
4. เลือกเครื่องสแกน (ชื่อจะขึ้นต้นด้วย "Barcode" หรือ "Scanner")
5. กด "จับคู่" (Pair)
6. ใส่รหัส PIN ถ้ามี (มักเป็น 0000 หรือ 1234)
7. **ทดสอบ:** สแกนบาร์โค้ดสินค้าดู

### 2. เครื่องพิมพ์ใบเสร็จ

#### แบบ USB
1. เสียบสาย USB เข้ากับคอมพิวเตอร์
2. เสียบสายไฟ และเปิดเครื่องพิมพ์
3. รอ Windows ติดตั้งไดร์เวอร์อัตโนมัติ

#### แบบ Network/Wi-Fi
1. เชื่อมต่อเครื่องพิมพ์กับ Wi-Fi ตามคู่มือของเครื่อง
2. จด IP Address ของเครื่องพิมพ์
3. ในโปรแกรม: ไปที่ **การตั้งค่า → เครื่องพิมพ์**
4. เลือก "เพิ่มเครื่องพิมพ์"
5. เลือกประเภท "Network Printer"
6. ใส่ IP Address และกด "เชื่อมต่อ"

#### ตั้งค่าในโปรแกรม
1. เปิดโปรแกรม BC Ai POS
2. ไปที่ **เมนู → การตั้งค่า → เครื่องพิมพ์**
3. เลือกเครื่องพิมพ์ที่เชื่อมต่อแล้ว
4. กด **"ทดสอบพิมพ์"**
5. ถ้าพิมพ์ออกมา แสดงว่าพร้อมใช้งาน ✅

### 3. ลิ้นชักเก็บเงิน (Cash Drawer)

1. เชื่อมต่อลิ้นชักกับเครื่องพิมพ์ใบเสร็จ
   - ใช้สาย RJ11 หรือ RJ12
   - เสียบที่ช่อง "Cash Drawer" หรือ "DK" บนเครื่องพิมพ์
2. เมื่อพิมพ์ใบเสร็จรับเงินสด ลิ้นชักจะเปิดอัตโนมัติ
3. **ทดสอบ:** ขายสินค้าทดสอบและชำระด้วยเงินสด

### 4. เครื่อง EDC รูดบัตร

1. เชื่อมต่อเครื่อง EDC ตามคู่มือของธนาคาร
2. เครื่อง EDC ทำงานแยกจากโปรแกรม POS
3. เมื่อลูกค้าชำระด้วยบัตร:
   - รูดบัตรที่เครื่อง EDC
   - รอ Approval
   - บันทึกยอดในโปรแกรม POS

## ⚙️ การตั้งค่าเบื้องต้น

### เช็คการเชื่อมต่ออินเทอร์เน็ต
1. เปิดโปรแกรม BC Ai POS
2. มุมบนขวาจะแสดงสัญญาณอินเทอร์เน็ต
3. ถ้าสีเขียว = เชื่อมต่อได้
4. ถ้าสีแดง = ไม่มีอินเทอร์เน็ต (ตรวจสอบ Wi-Fi/LAN)

### เลือกเครื่องพิมพ์เริ่มต้น
1. ไปที่ **เมนู → การตั้งค่า → เครื่องพิมพ์**
2. เลือกเครื่องพิมพ์หลัก
3. กด "บันทึก"

### ตั้งค่าขนาดหน้าจอ (ถ้าจำเป็น)
1. ไปที่ **เมนู → การตั้งค่า → หน้าจอ**
2. เลือกขนาดตัวอักษร (เล็ก/กลาง/ใหญ่)
3. เลือก Zoom level (80%-120%)

## ✅ ตรวจสอบความพร้อม

ก่อนเริ่มใช้งานจริง ตรวจสอบว่า:
- [ ] โปรแกรมเปิดได้
- [ ] เชื่อมต่ออินเทอร์เน็ตได้ (สัญญาณสีเขียว)
- [ ] เครื่องสแกนบาร์โค้ดทำงานได้
- [ ] เครื่องพิมพ์ใบเสร็จพิมพ์ออกได้
- [ ] ลิ้นชักเก็บเงินเปิดได้
- [ ] มีกระดาษใบเสร็จพอ
- [ ] มีเงินทอนเตรียมไว้

## ⚠️ ข้อควรระวัง

### ข้อกำหนดระบบ
- ✅ Windows 10 ขึ้นไป (แนะนำ Windows 11)
- ✅ Android 8.0 ขึ้นไป
- ✅ RAM อย่างน้อย 4GB
- ✅ Internet ความเร็วอย่างน้อย 5 Mbps

### อย่าทำ
- ❌ ห้ามติดตั้งบนคอมพิวเตอร์ที่ไม่มีสิทธิ์ (ต้องได้รับอนุญาตจากเจ้าของร้าน)
- ❌ ห้ามแชร์รหัสผ่านกับคนอื่น
- ❌ ห้ามใช้โปรแกรมเวอร์ชันเก่าหรือที่ดาวน์โหลดจากแหล่งอื่น

## 🆘 แก้ไขปัญหาเบื้องต้น

### โปรแกรมเปิดไม่ขึ้น
1. Restart คอมพิวเตอร์
2. ลองเปิดใหม่
3. ถ้ายังไม่ได้ ลงใหม่

### เชื่อมต่อเซิร์ฟเวอร์ไม่ได้
1. เช็คอินเทอร์เน็ต (ลองเปิดเว็บไซต์ดู)
2. ปิดโปรแกรม แล้วเปิดใหม่
3. ติดต่อผู้ดูแล IT

### เครื่องพิมพ์พิมพ์ไม่ออก
1. เช็คสายไฟเสียบดีหรือไม่
2. เช็คมีกระดาษหรือไม่
3. เช็คสาย USB/Network
4. ปิดเครื่อง รอ 10 วิ แล้วเปิดใหม่
