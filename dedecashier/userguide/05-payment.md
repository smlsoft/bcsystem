# การชำระเงน

##  ชองทางชำระเงน

### 1.  เงนสด
1. กด "เงนสด" หรอ `F8`
2. ระบุจำนวนเงินที่รับมา
3. ระบบคำนวณเงินทอน **+ เสียงพูด** 🔊
4. พิมพ์ใบเสร็จ + ลิ้นชักเปิดอัตโนมัติ

**ปมชวยคำนวณ:**
- **พอด** = ลกคาจายพอด
- **20/50/100/500/1000** = คลกธนบตรทไดรบ

### 2.  บตรเครดต/เดบต
1. กด "บตรเครดต"
2. ระบจำนวน (หรอ พอด)
3. **รดบตรทเครอง EDC**
4. รอ Approval (5-10 ว)
5. กด "ยนยน" เมอชำระสำเรจ
6. พมพใบเสรจ + สลปบตร

 **ระวง:** รอ EDC อนมตกอนยนยนทกครง

### 3.  QR Code
รองรบ: PromptPay, TrueMoney, ShopeePay, Alipay

1. กด "QR Code"
2. เลอกแอพ
3. **แสดง QR บนหนาจอ**
4. ลกคาสแกนดวยแอพธนาคาร
5. รอการแจงเตอน (10-30 ว)
6. มเสยง "ชำระเงนสำเรจ" 
7. กด "ยนยน"

 **เชคได:** หากรอนาน ใหลกคาดในแอพวาโอนสำเรจหรอยง

### 4.  คะแนนสะสม
1. กด "คะแนน"
2. ระบเบอรโทรหรอสแกนบตรสมาชก
3. ระบบแสดงคะแนนทม
4. ระบจำนวนคะแนนทใช
5. ยนยน

**อตราแลก:** 1 คะแนน = 1 บาท (ตามกำหนดราน)

### 5.  ชำระแบบผสม (Mixed Payment)
ชำระหลายชองทางในบลเดยว!

**ตวอยาง: ยอด 1,500**
1. ใชคะแนน 200 (เหลอ 1,300)
2. บตรเครดต 1,000 (เหลอ 300)
3. เงนสด 300 (ชำระครบ )

##  พมพใบเสรจ

### ใบเสรจมอะไรบาง?
-  รายการสนคา + ราคา
-  โปรโมชน/สวนลด
-  วธชำระเงน
-  เงนทอน (ถาเปนเงนสด)
-  คะแนนทไดรบ
-  QR Code รวว

### พมพซำ
1. เขา "ประวตการขาย"
2. เลอกบล
3. กด "พมพซำ" 

##  ยกเลกการชำระ
- **กอนยนยน:** กด "ยกเลก" หรอ `Esc`
- **หลงยนยนแลว:** ตองใหผจดการทำ "คนเงน/ยกเลกบล"

##  สำคญ!

-  เชคเงนทอนใหถกตอง
-  หามรบเงนกอนกดยนยน
-  หามปดโปรแกรมระหวางชำระเงน
