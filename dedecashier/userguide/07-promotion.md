## 🎁 ภาพรวมโปรโมชั่น

ระบบรองรับโปรโมชั่นหลากหลายรูปแบบเพื่อเพิ่มยอดขาย:

| ไอคอน | ประเภท | คำอธิบาย |
|------|--------|----------|
| 🏷️ | **ลดราคา** | ลดราคาสินค้าโดยตรง |
| 🎁 | **แถมฟรี** | ซื้อ 1 แถม 1, ซื้อครบแถมของ |
| 💯 | **แลกแต้ม** | ใช้แต้มสะสมแลกสินค้า/ส่วนลด |
| 🎫 | **คูปอง** | ใช้รหัสคูปองรับส่วนลด |
| 📦 | **เซ็ตสินค้า** | ซื้อครบเซ็ตได้ราคาพิเศษ |

---

## 🎯 ประเภทโปรโมชั่น

### 1️⃣ ลดราคาสินค้า (Discount)

#### 💰 ลดตามจำนวนเงิน
```
ตัวอย่าง:
สินค้า A ราคา 100 บาท
โปรโมชั่น: ลด 20 บาท
→ ลูกค้าจ่าย 80 บาท
```

#### 📊 ลดตามเปอร์เซ็นต์
```
ตัวอย่าง:
สินค้า B ราคา 500 บาท
โปรโมชั่น: ลด 10%
→ ลูกค้าจ่าย 450 บาท
```

#### 🎯 ลดตามจำนวนชิ้น
```
ตัวอย่าง:
ซื้อ 2 ชิ้น ลด 50 บาท
ซื้อ 3 ชิ้น ลด 100 บาท
```

---

### 2️⃣ แถมสินค้าฟรี (Buy X Get Y)

#### 🎁 ซื้อ 1 แถม 1
```
ตัวอย่าง:
ซื้อน้ำอัดลม 1 ขวด แถม 1 ขวด
→ สแกนสินค้า 1 ครั้ง ระบบเพิ่มอัตโนมัติ
```

#### 🎁 ซื้อ 2 แถม 1
```
ตัวอย่าง:
ซื้อแชมพู 2 ขวด แถม 1 ขวด
→ จ่าย 2 ขวด ได้ 3 ขวด
```

#### 🎁 ซื้อครบแถมของ
```
ตัวอย่าง:
ซื้อครบ 500 บาท แถมถุงผ้า
→ ระบบเตือนอัตโนมัติ
```

---

### 3️⃣ เซ็ตสินค้า (Bundle)

#### 📦 เซ็ตราคาพิเศษ
```
ตัวอย่าง:
ข้าว + ไก่ทอด + น้ำ = 99 บาท
(ปกติรวม 120 บาท)
```

#### 📦 เซ็ตตามประเภท
```
ตัวอย่าง:
เลือกขนม 3 ชิ้น ราคา 50 บาท
(จากขนมที่กำหนด)
```

---

### 4️⃣ แลกแต้มสะสม (Point Redemption)

#### ⭐ ใช้แต้มแลกสินค้า
```
ตัวอย่าง:
สมาชิกมีแต้ม 1,000 คะแนน
แก้วน้ำ = 500 คะแนน
→ แลกได้ 2 แก้ว
```

#### ⭐ ใช้แต้มแลกส่วนลด
```
ตัวอย่าง:
100 คะแนน = ส่วนลด 10 บาท
มีแต้ม 500 คะแนน = ลดได้ 50 บาท
```

---

### 5️⃣ คูปองส่วนลด (Coupon)

#### 🎫 รหัสคูปอง
```
ตัวอย่าง:
SAVE50 = ลด 50 บาท
SALE10 = ลด 10%
```

#### 🎫 คูปองตามยอดซื้อ
```
ตัวอย่าง:
ซื้อครบ 1,000 บาท ลด 100 บาท
```

---

## 🔧 การใช้งานโปรโมชั่น

### ✅ โปรโมชั่นอัตโนมัติ

ระบบจะคำนวณและใช้โปรโมชั่นอัตโนมัติเมื่อ:

1. **สแกนสินค้า** ที่มีโปรโมชั่น
2. **ครบเงื่อนไข** ตามโปรโมชั่น (จำนวน/ยอดเงิน)
3. **แสดงเตือน** ⏳ กำลังจะได้โปรโมชั่น หรือ ✅ ได้โปรโมชั่นแล้ว

#### 📱 หน้าจอแสดงโปรโมชั่น

```
┌─────────────────────────────────┐
│  🎉 โปรโมชั่น                   │
├─────────────────────────────────┤
│                                 │
│  ⏳ ซื้อ 1 แถม 1               │
│     → ต้องการอีก 1 ชิ้น         │
│                                 │
│  ✅ ลดราคา 10%                 │
│     → ประหยัดไป 50 บาท         │
│                                 │
└─────────────────────────────────┘
```

**คำอธิบายสัญลักษณ์:**
- ⏳ = **กำลังจะได้** (ยังไม่ครบเงื่อนไข)
- ✅ = **ได้แล้ว** (ใช้โปรโมชั่นแล้ว)

---

### 🎯 การใช้คูปอง

#### วิธีใช้คูปอง:

1. **สแกนสินค้า** ให้ครบตามปกติ
2. กดปุ่ม **"คูปอง"** หรือ **F7**
3. **กรอกรหัสคูปอง** หรือสแกน QR Code คูปอง
4. ระบบจะ**คำนวณส่วนลด**อัตโนมัติ
5. ตรวจสอบส่วนลดที่ได้
6. **ชำระเงิน** ตามยอดหลังหักส่วนลด

#### ตัวอย่าง:

```
สินค้ารวม:           1,000 บาท
ใช้คูปอง SAVE100:     -100 บาท
────────────────────────────────
ยอดชำระ:              900 บาท
```

---

### ⭐ การแลกแต้มสะสม

#### วิธีแลกแต้ม:

1. **ระบุสมาชิก** (สแกนบัตร หรือกรอกเบอร์โทร)
2. ระบบแสดง**แต้มคงเหลือ**
3. กดปุ่ม **"แลกแต้ม"** หรือ **F8**
4. **เลือกของแลก** หรือ **ระบุจำนวนแต้มที่ใช้**
5. ระบบหัก**แต้มอัตโนมัติ**
6. แสดง**ยอดชำระหลังใช้แต้ม**

#### ตัวอย่างแลกสินค้า:

```
┌─────────────────────────────────┐
│  ⭐ แต้มคงเหลือ: 1,500 คะแนน   │
├─────────────────────────────────┤
│  📦 ของแลก:                     │
│                                 │
│  [1] แก้วน้ำ       500 คะแนน   │
│  [2] ร่ม           800 คะแนน   │
│  [3] เสื้อยืด     1,200 คะแนน   │
│  [4] กระเป๋า      1,500 คะแนน   │
│                                 │
└─────────────────────────────────┘
```

#### ตัวอย่างแลกส่วนลด:

```
สินค้ารวม:           1,000 บาท
แต้มคงเหลือ:           500 คะแนน
ใช้แต้ม 500 คะแนน:     -50 บาท
────────────────────────────────
ยอดชำระ:              950 บาท
แต้มคงเหลือ:             0 คะแนน
```

## 🎊 เคล็ดลับการใช้โปรโมชั่น

### ✨ เพิ่มยอดขายด้วยโปรโมชั่น

#### 1. **แจ้งลูกค้า** ให้รู้ว่ามีโปรโมชั่น
```
"คุณลูกค้า ตอนนี้ซื้อ 2 แถม 1 นะคะ
ต้องการเพิ่มอีกไหมคะ?"
```

#### 2. **ดูการแจ้งเตือน** ⏳ กำลังจะได้โปรโมชั่น
```
⏳ ซื้อ 1 แถม 1 → ต้องการอีก 1 ชิ้น
→ แจ้งลูกค้าให้เพิ่มสินค้า
```

#### 3. **ตรวจสอบโปรโมชั่น** หลังสแกนครบ
```
ดูที่หน้าจอว่ามี ✅ โปรโมชั่นใช้งานหรือไม่
→ ยืนยันกับลูกค้า
```

---

### 🔍 ตรวจสอบโปรโมชั่น

#### ก่อนชำระเงิน ให้เช็ค:

- ✅ โปรโมชั่นใช้งานครบหรือไม่?
- ✅ ส่วนลดถูกต้องไหม?
- ✅ แต้มสะสมได้ครบไหม?
- ✅ ลูกค้าได้รับของแถมครบไหม?

---

### ⚠️ ข้อควรระวัง

| สถานการณ์ | วิธีจัดการ |
|-----------|-----------|
| **โปรโมชั่นไม่ทำงาน** | ตรวจสอบเงื่อนไข (วันเวลา, จำนวน) |
| **คูปองใช้ไม่ได้** | เช็คว่าคูปองหมดอายุหรือไม่ |
| **แต้มไม่พอ** | แจ้งลูกค้าแต้มที่มี และที่ต้องการ |
| **สินค้าไม่ร่วมโปร** | แจ้งลูกค้าว่าสินค้าไม่อยู่ในเงื่อนไข |

## 🖨️ โปรโมชั่นในใบเสร็จ

โปรโมชั่นทุกประเภทจะแสดงในใบเสร็จอัตโนมัติ:

```
================================
       ใบเสร็จรับเงิน
================================

น้ำอัดลม x2        60.00
  🎁 แถม 1         -30.00

ขนมขบเคี้ยว        100.00
  🏷️ ลด 10%        -10.00

--------------------------------
รวม                 120.00
ส่วนลดคูปอง SAVE20  -20.00
แต้มสะสม 500 แต้ม   -50.00
================================
ยอดชำระ             50.00
================================

🎉 โปรโมชั่นที่ใช้:
  • ซื้อ 1 แถม 1 (น้ำอัดลม)
  • ลด 10% (ขนมขบเคี้ยว)
  • คูปอง SAVE20
  • แลกแต้ม 500 คะแนน

⭐ แต้มสะสมครั้งนี้: +50 คะแนน
💯 แต้มคงเหลือ: 50 คะแนน
```

## ❓ คำถามที่พบบ่อย

**Q: โปรโมชั่นไม่ทำงานอัตโนมัติ?**  
A: ตรวจสอบ:
- เงื่อนไขโปรโมชั่น (วันเวลา, สาขา)
- จำนวนสินค้าครบตามเงื่อนไขหรือไม่
- โปรโมชั่นเปิดใช้งานอยู่หรือไม่

**Q: ใช้คูปองและแลกแต้มพร้อมกันได้ไหม?**  
A: ได้! ระบบรองรับการใช้หลายโปรโมชั่นพร้อมกัน

**Q: แลกแต้มแล้วยกเลิกบิล แต้มจะคืนไหม?**  
A: คืนอัตโนมัติภายใน 24 ชั่วโมง

**Q: ลูกค้าไม่มีบัตรสมาชิก ใช้แต้มได้ไหม?**  
A: กรอกเบอร์โทรศัพท์แทนได้
