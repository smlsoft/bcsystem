│  🍔 อาหาร        │  [3] กาแฟ        45 x 1 = 45│

│  🍰 ขนม          │  ────────────────────────────│
│  📱 อิเล็กฯ      │  รวม:            115 บาท    │
│  👕 เสื้อผ้า      │  ส่วนลด:           0 บาท    │
│                  │  ────────────────────────────│
│  [รูปสินค้า]     │  💰 ยอดชำระ:    115 บาท    │

│  [รูปสินค้า]     │  [ชำระเงิน F8]              │

| ส่วน | คำอธิบาย |
|------|----------|

| **ขวา** | รายการสินค้าในบิล + ยอดรวม |
| **ล่าง** | ปุ่มฟังก์ชัน + Numpad |

## 🎯 3 วิธีเพิ่มสินค้าเข้าตะกร้า

### 1️⃣ สแกนบาร์โค้ด (เร็วที่สุด ⚡)

**ขั้นตอน:**
1. จ่อบาร์โค้ดที่เครื่องสแกน
2. ได้ยินเสียง **"บี๊บ"** = สแกนสำเร็จ
3. สินค้าเพิ่มอัตโนมัติทันที ✅

**💡 เคล็ดลับ:**
- ✅ สแกนหลายชิ้นติดกัน **ไม่ต้องรอ**
- ✅ มี **Overlay แสดงบาร์โค้ด** 5 วินาที
- ✅ ถ้าต้องการหลายชิ้น: กด **จำนวน** → ระบุ → สแกน

**ตัวอย่าง:**
```
สแกน: 8850123456789
→ เพิ่ม "น้ำดื่ม" 1 ขวด (20 บาท)

กด: 10 + สแกน: 8850123456789  
→ เพิ่ม "น้ำดื่ม" 10 ขวด (200 บาท)
```

**🔗 รายละเอียดเพิ่มเติม:**
- ดู [06-barcode.md](06-barcode.md) - 5 วิธีการสแกนบาร์โค้ด

---

### 2️⃣ ค้นหาสินค้า

**ขั้นตอน:**
1. คลิกช่องค้นหา หรือกด **`F2`**
2. พิมพ์ชื่อสินค้าหรือรหัส (พิมพ์บางส่วนก็ได้)
3. เลือกจากรายการ → กด **`Enter`**

**ตัวอย่าง:**
```
พิมพ์: "น้ำ"
→ แสดงรายการ:
   [1] น้ำดื่ม 600ml
   [2] น้ำอัดลม
   [3] น้ำส้มคั้น

กด Enter หรือคลิก
→ เพิ่มสินค้าเข้าบิล
```

**💡 เทคนิค:**
- พิมพ์ **ชื่อย่อ** ก็ค้นหาได้ (เช่น "กฟ" → "กาแฟ")
- ค้นหาด้วย **บาร์โค้ด** ก็ได้
- รองรับ **ภาษาไทยและอังกฤษ**

---

### 3️⃣ เลือกจากเมนู

**ขั้นตอน:**
1. คลิก**หมวดหมู่สินค้า**ด้านซ้าย (เช่น 🥤 เครื่องดื่ม)
2. คลิก**รูปสินค้า**ที่ต้องการ
3. ระบุจำนวน (ถ้าต้องการเปลี่ยน)
4. กด **"เพิ่ม"**

**ตัวอย่าง:**
```
คลิก: 🥤 เครื่องดื่ม
→ แสดงรูปเครื่องดื่มทั้งหมด

คลิก: รูปน้ำดื่ม
→ เพิ่มเข้าบิล 1 ขวด
```

**💡 เหมาะกับ:**
- ลูกค้าไม่รู้ชื่อสินค้า (ชี้รูปได้)
- พนักงานใหม่ (ดูรูปง่ายกว่า)
- จอสัมผัส (Touch Screen)

## 🤖 AI แนะนำสินค้า

เมื่อเพิ่มสินค้า **AI จะแนะนำสินค้าที่เข้ากัน**:

### ตัวอย่างการแนะนำ

```
┌─────────────────────────────────┐
│  💡 AI แนะนำ                     │
├─────────────────────────────────┤
│                                 │
│  คุณซื้อ: กาแฟ                  │
│                                 │
│  🍰 ลูกค้าคนอื่นซื้อเพิ่ม:      │
│     • เค้กช็อกโกแลต (89%)       │
│     • คุกกี้ (76%)               │
│     • โดนัท (65%)                │
│                                 │
│  [คลิกเพื่อเพิ่มทันที]          │
└─────────────────────────────────┘
```

### ประเภทการแนะนำ

| สถานการณ์ | แนะนำ |
|-----------|-------|
| ซื้อ**กาแฟ** | ☕ → 🍰 เค้ก |
| ซื้อ**เสื้อ** | 👕 → 👖 กางเกง |
| ซื้อ**แชมพู** | 🧴 → 🧼 ครีมนวด |
| ซื้อ**นม** | 🥛 → 🍞 ขนมปัง |

**💡 ข้อดี:**
- 📈 เพิ่มยอดขาย
- 😊 ลูกค้าพอใจ (ไม่ลืมของที่ต้องการ)
- ⚡ รวดเร็ว (คลิกเพิ่มทันที)

## ✏️ แก้ไข/ลบสินค้า

### เปลี่ยนจำนวน

**3 วิธี:**

#### 1. กดปุ่ม +/-
```
คลิกที่แถวสินค้า
→ กดปุ่ม "+" เพิ่ม 1
→ กดปุ่ม "-" ลด 1
```

#### 2. พิมพ์จำนวนใหม่
```
คลิกที่จำนวน
→ พิมพ์เลขใหม่ (เช่น 10)
→ กด Enter
```

#### 3. ดับเบิลคลิก
```
ดับเบิลคลิกที่แถวสินค้า
→ แก้ไขจำนวนได้เลย
```

---

### ลบสินค้า

**วิธีที่ 1: ปุ่มลบ**
```
1. คลิกแถวสินค้า (ไฮไลท์สีฟ้า)
2. กดปุ่ม "ลบ" หรือ `Delete`
3. ยืนยันการลบ → ตกลง
```

**วิธีที่ 2: กดขวา**
```
คลิกขวาที่แถวสินค้า
→ เลือก "ลบสินค้า"
→ ยืนยัน
```

**⚠️ คำเตือน:**
- ตรวจสอบก่อนลบทุกครั้ง
- ถ้าลบผิด → กด `Ctrl+Z` ย้อนกลับได้

---

## 💰 ส่วนลด

### 1️⃣ ส่วนลดรายการ (Per Item)

**ขั้นตอน:**
1. คลิกแถวสินค้า
2. กดปุ่ม **"ส่วนลด"** 💸
3. เลือก **%** หรือ **บาท**
4. ระบุจำนวน → ตกลง

**ตัวอย่าง:**
```
สินค้า: กาแฟ 50 บาท

ลด 10%:
→ ลด 5 บาท
→ เหลือ 45 บาท

ลด 10 บาท:
→ เหลือ 40 บาท
```

---

### 2️⃣ ส่วนลดท้ายบิล (Total Discount)

**ขั้นตอน:**
1. เพิ่มสินค้าครบแล้ว
2. กดปุ่ม **"ส่วนลด"** (ด้านล่าง)
3. เลือกประเภท:
   - 💵 **บาท** - ลดจำนวนเงิน
   - 📊 **%** - ลดเปอร์เซ็นต์
4. ระบุมูลค่า → ตกลง

**ตัวอย่าง:**
```
ยอดรวม: 1,000 บาท

ลด 10%:
→ ลด 100 บาท
→ ชำระ 900 บาท

ลด 50 บาท:
→ ชำระ 950 บาท
```

---

### 3️⃣ โปรโมชั่นอัตโนมัติ 🎉

**ระบบคำนวณโปรที่ดีที่สุดให้เอง!**

```
ไม่ต้องเลือก ไม่ต้องคิด
→ ระบบจะใช้โปรที่ดีที่สุดอัตโนมัติ

ตัวอย่าง:
• ซื้อ 2 แถม 1 ✅
• ลดราคา 10% ✅
• คูปอง 50 บาท ✅

→ ได้ส่วนลดทั้งหมดอัตโนมัติ!
```

## 📝 พักบิล (Hold)

### บันทึกบิลรอ

**เมื่อไหร่ต้องใช้:**
- ลูกค้าลืมเงิน 💸
- ลูกค้าต้องการเพิ่มสินค้า 🛒
- ต้องรับลูกค้าคนอื่นก่อน ⏰

**ขั้นตอน:**
1. เพิ่มสินค้าเข้าตะกร้า
2. กดปุ่ม **"พักบิล"** 📁 หรือ **`F3`**
3. ระบุชื่อลูกค้า/โต๊ะ (ไม่บังคับ)
4. กด **"บันทึก"**

**ตัวอย่าง:**
```
┌─────────────────────────────────┐
│  💾 บันทึกบิลรอ                  │
├─────────────────────────────────┤
│                                 │
│  ชื่อลูกค้า: คุณสมชาย           │
│  หมายเหตุ: โต๊ะ 5                │
│  ยอดรวม: 250 บาท               │
│                                 │
│  [บันทึก] [ยกเลิก]              │
└─────────────────────────────────┘

→ บิลถูกบันทึกแล้ว ✅
→ เริ่มบิลใหม่ได้เลย
```

---

### เรียกบิลรอกลับมา

**ขั้นตอน:**
1. กดปุ่ม **"บิลรอ"** 📋 หรือ **`F4`**
2. เลือกบิลที่ต้องการ
3. กดปุ่ม **"เปิดบิล"**
4. ชำระเงินต่อได้เลย

**ตัวอย่างรายการบิลรอ:**
```
┌─────────────────────────────────┐
│  📋 บิลรอทั้งหมด                 │
├─────────────────────────────────┤
│                                 │
│  [1] คุณสมชาย - โต๊ะ 5          │
│      250 บาท | 14:30             │
│                                 │
│  [2] คุณสมหญิง                   │
│      150 บาท | 14:45             │
│                                 │
│  [3] โต๊ะ 8                       │
│      450 บาท | 15:00             │
│                                 │
│  [เปิดบิล] [ลบบิล]               │
└─────────────────────────────────┘
```

**💡 เคล็ดลับ:**
- พักบิลได้**ไม่จำกัด**จำนวน
- บิลรอจะ**ไม่หายไป** (เก็บไว้ได้นาน)
- ถ้าไฟดับ → **บิลรอยังอยู่**

## 💡 เคล็ดลับการขาย

### เพิ่มความเร็ว ⚡

1. **ใช้ปุ่มลัด** แทนการคลิก
   ```
   F2 = ค้นหา (เร็วกว่าคลิก)
   F8 = ชำระเงิน (ไม่ต้องเลื่อนหา)
   ```

2. **สแกนบาร์โค้ด** แทนการค้นหา
   ```
   สแกน = 1 วินาที
   ค้นหา = 5 วินาที
   ```

3. **จำรหัสสินค้า** ที่ขายบ่อย
   ```
   พิมพ์: 001 + Enter
   → น้ำดื่ม (ไม่ต้องค้นหา)
   ```

---

### ลดข้อผิดพลาด ✅

1. **ตรวจสอบก่อนกด Enter** ทุกครั้ง
2. **ยืนยันกับลูกค้า** ก่อนชำระเงิน
3. **นับเงิน 2 ครั้ง** (รับ + ทอน)
4. **เก็บใบเสร็จเรียบร้อย**

---

### จัดการบิลรอ 📋

```
แนะนำ:
• ระบุชื่อลูกค้าหรือเลขโต๊ะเสมอ
• ตรวจสอบบิลรอก่อนปิดกะ
• ลบบิลรอที่เก่าเกิน 1 วัน
```
