## 🎯 ปุ่มฟังก์ชัน F1-F12

### การใช้งานหลัก

| ปุ่ม | ฟังก์ชัน | คำอธิบาย |
|-----|---------|----------|
| **F1** | 📖 **ช่วยเหลือ** | เปิดคู่มือการใช้งาน |
| **F2** | 🔍 **ค้นหาสินค้า** | เปิดหน้าจอค้นหาสินค้า |
| **F3** | 📱 **ลูกค้า** | ค้นหา/เพิ่มลูกค้า |
| **F4** | ✏️ **แก้ไข** | แก้ไขรายการที่เลือก |
| **F5** | 🔄 **รีเฟรช** | รีเฟรชหน้าจอ |
| **F6** | 📄 **บิลเดิม** | ดูบิลที่ขายไปแล้ว |
| **F7** | 🎫 **คูปอง** | ใช้คูปองส่วนลด |
| **F8** | ⭐ **แลกแต้ม** | แลกแต้มสะสม |
| **F9** | 🎉 **โปรโมชั่น** | ดูโปรโมชั่นทั้งหมด |
| **F10** | 👥 **สมาชิก** | จัดการสมาชิก |
| **F11** | 🔄 **ซิงค์** | ซิงค์ข้อมูลกับเซิร์ฟเวอร์ |
| **F12** | ⚙️ **ตั้งค่า** | เปิดหน้าตั้งค่า |

---

### 📖 F1: ช่วยเหลือ

```
เปิดคู่มือการใช้งาน
→ แสดงหน้าจอช่วยเหลือ
→ ค้นหาวิธีใช้งานได้ทันที
```

**การใช้งาน:**
- กด **F1** ทุกเมื่อที่ต้องการความช่วยเหลือ
- แสดงคู่มือตามหน้าจอที่กำลังใช้งาน

---

### 🔍 F2: ค้นหาสินค้า

```
เปิดหน้าจอค้นหาสินค้าอย่างรวดเร็ว
→ พิมพ์ชื่อสินค้า/บาร์โค้ด
→ กด Enter เพิ่มเข้าบิล
```

**เคล็ดลับ:**
- พิมพ์**ชื่อบางส่วน**ก็ค้นหาได้
- รองรับ**การสแกนบาร์โค้ด**

---

### 📱 F3: ลูกค้า

```
ค้นหาหรือเพิ่มข้อมูลลูกค้า
→ พิมพ์เบอร์โทร/ชื่อ
→ เพิ่มลูกค้าใหม่ได้ทันที
```

---

### ✏️ F4: แก้ไข

```
แก้ไขรายการที่เลือก
→ เปลี่ยนจำนวน
→ เปลี่ยนราคา (ต้องมีสิทธิ์)
→ เปลี่ยนหมายเหตุ
```

**ขั้นตอน:**
1. คลิกเลือกรายการในบิล
2. กด **F4**
3. แก้ไขข้อมูล
4. กด Enter บันทึก

---

### 🔄 F5: รีเฟรช

```
รีเฟรชหน้าจอและข้อมูล
→ โหลดข้อมูลใหม่
→ แก้ปัญหาจอค้าง
```

---

### 📄 F6: บิลเดิม

```
ดูบิลที่ขายไปแล้ว
→ ค้นหาบิล
→ พิมพ์ใบเสร็จซ้ำ
→ คืนสินค้า
```

**การใช้งาน:**
1. กด **F6**
2. เลือกวันที่/เลขบิล
3. เลือกบิลที่ต้องการ
4. พิมพ์ซ้ำหรือคืนสินค้า

---

### 🎫 F7: คูปอง

```
ใช้คูปองส่วนลด
→ พิมพ์รหัสคูปอง
→ หรือสแกน QR Code
→ ระบบคำนวณส่วนลดอัตโนมัติ
```

---

### ⭐ F8: แลกแต้ม

```
ใช้แต้มสะสม
→ แลกของรางวัล
→ หรือแลกส่วนลด
```

**เงื่อนไข:**
- ต้องระบุสมาชิกก่อน
- มีแต้มเพียงพอ

---

### 🎉 F9: โปรโมชั่น

```
ดูรายการโปรโมชั่นทั้งหมด
→ โปรโมชั่นที่ใช้อยู่
→ เงื่อนไข
→ ของแถม
```

---

### 👥 F10: สมาชิก

```
จัดการสมาชิก
→ เพิ่มสมาชิกใหม่
→ แก้ไขข้อมูล
→ เติมเงินฝาก
→ ดูประวัติการซื้อ
```

---

### 🔄 F11: ซิงค์ข้อมูล

```
ซิงค์ข้อมูลกับเซิร์ฟเวอร์
→ อัปเดตสินค้าใหม่
→ อัปเดตโปรโมชั่น
→ อัปเดตราคา
→ ส่งบิลขึ้นเซิร์ฟเวอร์
```

**เมื่อไหร่ต้องกด:**
- เริ่มกะใหม่
- เพิ่มสินค้าใหม่
- เปลี่ยนราคา
- อินเทอร์เน็ตกลับมาใหม่

---

### ⚙️ F12: ตั้งค่า

```
เปิดหน้าตั้งค่าระบบ
→ ตั้งค่าเครื่องพิมพ์
→ ตั้งค่าการชำระเงิน
→ ตั้งค่าหน้าจอ
→ ตั้งค่าผู้ใช้งาน
```

## 🛒 ปุ่มลัดการขาย

### ปุ่มลัดในหน้าขาย

| ปุ่มลัด | ฟังก์ชัน |
|---------|---------|
| **Enter** | ✅ ยืนยัน/ค้นหา |
| **Esc** | ❌ ยกเลิก/ย้อนกลับ |
| **Del** | 🗑️ ลบรายการที่เลือก |
| **Tab** | ➡️ เปลี่ยนฟิลด์ถัดไป |
| **↑ ↓** | ⬆️⬇️ เลือกรายการในลิสต์ |
| **Page Up** | ⬆️ เลื่อนขึ้น |
| **Page Down** | ⬇️ เลื่อนลง |

---

### 🛒 การใช้ Enter

```
1. พิมพ์บาร์โค้ด → Enter = เพิ่มสินค้า
2. พิมพ์ชื่อสินค้า → Enter = ค้นหา
3. พิมพ์เบอร์โทร → Enter = ค้นหาสมาชิก
4. พิมพ์จำนวน → Enter = เปลี่ยนจำนวน
```

---

### 🗑️ การใช้ Del

```
1. คลิกเลือกรายการในบิล
2. กด Del
3. ยืนยันการลบ
→ รายการถูกลบออก
```

## 💳 ปุ่มลัดการชำระเงิน

### ปุ่มลัดในหน้าชำระเงิน

| ปุ่มลัด | ช่องทางชำระ |
|---------|-------------|
| **1** | 💵 เงินสด |
| **2** | 💳 บัตรเครดิต |
| **3** | 📱 QR Code (PromptPay) |
| **4** | 💰 เงินโอน |
| **5** | 🎫 คูปอง |
| **6** | ⭐ แต้มสะสม |
| **7** | 💳 บัตรสมาชิกเงินฝาก |

---

### 💡 ตัวอย่างการใช้งาน

#### ชำระเงินสดอย่างรวดเร็ว:

```
1. สแกนสินค้าเสร็จ
2. กด Enter (ไปหน้าชำระเงิน)
3. กด 1 (เลือกเงินสด)
4. พิมพ์จำนวนเงินที่รับ (เช่น 1000)
5. กด Enter
→ ระบบคำนวณเงินทอน
→ พิมพ์ใบเสร็จอัตโนมัติ
```

#### ชำระด้วย QR Code:

```
1. กด Enter (ไปหน้าชำระเงิน)
2. กด 3 (เลือก QR Code)
3. แสดง QR Code ให้ลูกค้าสแกน
4. รอการชำระเงิน
5. ระบบยืนยันอัตโนมัติ
```

## 🔢 ปุ่มตัวเลข Numpad

### การใช้ Numpad ในการขาย

| การกด | ผลลัพธ์ |
|-------|---------|
| **5** + Enter | ค้นหาสินค้ารหัส 5 |
| **10** + **\*** + สแกน | ซื้อ 10 ชิ้น |
| **2.5** + **\*** + สแกน | ซื้อ 2.5 กิโลกรัม |

---

### 🎯 quantity * barcode Format

```
ตัวอย่าง: ต้องการซื้อน้ำ 10 ขวด

วิธีที่ 1: พิมพ์จำนวนก่อน
10 * [สแกนบาร์โค้ด]

วิธีที่ 2: สแกนแล้วแก้จำนวน
[สแกนบาร์โค้ด] → F4 → แก้จำนวนเป็น 10

วิธีที่ 3: สแกนหลายครั้ง
[สแกน] [สแกน] [สแกน] ... (10 ครั้ง)
```

## 💡 เคล็ดลับการใช้ปุ่มลัด

### 🎯 เพิ่มความเร็ว 2 เท่า!

#### 1. **จำปุ่ม F ที่ใช้บ่อย**
```
F2 = ค้นหาสินค้า (ใช้บ่อยสุด!)
F4 = แก้ไข
F6 = บิลเดิม
F8 = แลกแต้ม
```

#### 2. **ใช้ Numpad แทนการคลิก**
```
พิมพ์จำนวน + * + สแกน
= เร็วกว่าการแก้ไขทีหลัง!
```

#### 3. **จำปุ่มชำระเงิน**
```
1 = เงินสด (ใช้บ่อยสุด)
3 = QR Code (ใช้บ่อยรองลงมา)
```

#### 4. **ใช้ Enter ให้คุ้ม**
```
Enter = ยืนยันทุกอย่าง
ไม่ต้องคลิกปุ่ม "OK" เลย!
```

## 🎓 ฝึกฝนการใช้ปุ่มลัด

### แบบฝึกหัด

ลองทำตามขั้นตอนนี้โดยใช้**ปุ่มลัดเท่านั้น**:

```
1. เปิดบิลใหม่ (Ctrl+N)
2. ค้นหาสินค้า "น้ำดื่ม" (F2)
3. เพิ่ม 5 ขวด (5 * Enter)
4. ค้นหาสมาชิก "081-234-5678" (F3)
5. ชำระเงินสด 100 บาท (Enter → 1 → 100 → Enter)
```

**เป้าหมาย:** ทำให้เสร็จภายใน **15 วินาที!**
